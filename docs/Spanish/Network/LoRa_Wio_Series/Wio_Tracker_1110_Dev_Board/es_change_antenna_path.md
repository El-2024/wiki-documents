La placa de desarrollo Wio Tracker 1110 utiliza por defecto la antena LoRa integrada. Sin embargo, para obtener una mejor señal, puede ser necesario usar una antena LoRa externa. Este capítulo guía al usuario para cambiar la ruta de la antena y conectar una antena externa.

* Retira la resistencia `R3`.
* Cambia la resistencia `R4` por un puente de 0 ohmios (`0R`).

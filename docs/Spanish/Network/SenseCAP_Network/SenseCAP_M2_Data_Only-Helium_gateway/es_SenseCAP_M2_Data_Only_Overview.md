**¿Qué es SenseCAP M2 Data Only?**

SenseCAP M2 Data Only es una puerta de enlace LoRaWAN conectada a la red Helium LongFi, capaz de proporcionar cobertura inalámbrica de varios kilómetros y capacidad de transmisión de datos para dispositivos de Largo Alcance / LoRaWAN.

Como puerta de enlace LoRaWAN, puede proporcionar cobertura inalámbrica de varios kilómetros y gran capacidad de transmisión de datos para dispositivos de Largo Alcance / LoRaWAN.

Al ser una gateway compatible con la red Helium LongFi, puede ayudarte a construir una red local LoRaWAN y desarrollar diversas aplicaciones LoRaWAN a bajo costo, mientras disfrutas de los beneficios de formar parte de la Red del Pueblo.

SenseCAP M2 está disponible en versiones US915 y EU868 con certificaciones FCC y CE. Pronto estarán disponibles más versiones con diferentes bandas de frecuencia y certificaciones para otros países y regiones.

**Características**
===================

![](https://www.sensecapmx.com/wp-content/uploads/2022/07/sensecap-m2-DO-features.png)

**· Cobertura Amplia y Señal Potente**

Con una antena de alto rendimiento de 3dBi, SenseCAP M2 puede proporcionar cobertura LoRaWAN de varios kilómetros y una señal potente. Además, puedes actualizar la antena según el entorno y las necesidades de uso.

**· Rendimiento Excelente y Estable**

SenseCAP M2 está equipado con la solución de hardware madura MT7628 y el chip de banda base Long Range Semtech SX1302/SX1303. Cuenta con BLE integrado y soporta conexión a Internet tanto por Wi-Fi como por Ethernet. También incluye PoE (Power over Ethernet) para alimentar el dispositivo sin necesidad de una fuente externa.

**· Herramientas Profesionales de Gestión y Servicios en la Nube**

Los usuarios pueden configurar fácilmente el hotspot con la App SenseCAP Hotspot en unos simples pasos. También se incluyen el Panel de Control SenseCAP y la Consola Local para monitorear y gestionar el Hotspot de forma fácil y eficiente.

**· Soporte Técnico en Línea Eficiente y Oportuno**

Además de las herramientas profesionales y los servicios en la nube, SenseCAP es ampliamente reconocido por su soporte técnico. Siempre tendrás a disposición asistencia técnica rápida y efectiva para resolver cualquier problema.

**· Seguro y Confiable**

En una puerta de enlace que gestiona grandes volúmenes de datos, la seguridad es fundamental. SenseCAP M2 Data Only garantiza autenticación segura y conectividad confiable a la Red del Pueblo.

**· Instalación Fácil en Pocos Pasos**

Listo para usarse con la App SenseCAP Hotspot en unos simples pasos.

**· Accesorios Variados**

Además de las poderosas funciones del dispositivo, SenseCAP ofrece una variedad de accesorios para maximizar el rendimiento de los Hotspots: antenas IoT de alto rendimiento, cables de antena, soportes de montaje y protectores contra rayos.

**Nota:**

* La ranura para tarjeta MicroSD está reservada para uso futuro. No se necesita MicroSD para usar SenseCAP M2 como gateway LoRaWAN.

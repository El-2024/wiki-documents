**¿Qué es SenseCAP M2 Solo Datos?**

SenseCAP M2 Solo Datos es un gateway LoRaWAN conectado a la red Helium LongFi, capaz de proporcionar cobertura inalámbrica de varios kilómetros y capacidad de transmisión de datos para dispositivos de largo alcance / LoRaWAN.

Como gateway LoRaWAN, puede ofrecer cobertura de red inalámbrica de largo alcance y transmisión de datos para dispositivos LoRaWAN.

Como gateway compatible con la red Helium LongFi, te permite construir una red LoRaWAN local y desarrollar diversas aplicaciones LoRaWAN a bajo costo, mientras disfrutas de los beneficios de ser parte de la red del pueblo (People’s Network).

Actualmente, SenseCAP M2 está disponible en versiones US915 y EU868 con certificados FCC y CE. Próximamente estarán disponibles más versiones con diferentes bandas de frecuencia y certificados correspondientes para varios países y regiones.

**Características**
===================

![](https://www.sensecapmx.com/wp-content/uploads/2022/07/sensecap-m2-DO-features.png)

**· Cobertura de largo alcance y señal potente**

Con una antena de alto rendimiento de 3dBi, SenseCAP M2 puede proporcionar cobertura LoRaWAN de varios kilómetros y una señal fuerte. Siempre puedes mejorar la antena según el entorno y tus necesidades.

**· Rendimiento excelente y estable**

SenseCAP M2 está basado en la solución de hardware madura MT7628 y el chip baseband de largo alcance Semtech SX1302/SX1303. Incluye BLE integrado y es compatible con conexión a Internet por Wi-Fi y Ethernet. Para quienes desean alimentar el dispositivo por Ethernet en lugar de usar un cable de energía adicional, también incluye la función PoE.

**· Herramientas de gestión profesional y servicios en la nube**

Los usuarios pueden configurar el hotspot en unos pocos pasos usando la app SenseCAP Hotspot. También se han desarrollado el SenseCAP Hotspot Dashboard y la Consola Local SenseCAP para monitorear y administrar el dispositivo de manera fácil y eficiente.

**· Soporte en línea eficiente y oportuno**

Además de las herramientas de gestión y servicios en la nube, SenseCAP ha sido ampliamente elogiado por su soporte técnico. Cuando surjan problemas técnicos, el soporte en línea estará disponible para ayudarte de forma rápida y eficaz.

**· Seguro y confiable**  
Para un gateway que maneja grandes volúmenes de transmisión de datos, la seguridad es una prioridad. SenseCAP M2 Solo Datos garantiza autenticación segura y conectividad confiable con la red del pueblo.

**· Configuración fácil en pocos pasos**

Listo para usarse con la app SenseCAP Hotspot de manera sencilla.

**· Accesorios variados**

Además de sus potentes funciones, SenseCAP ofrece una amplia variedad de accesorios para maximizar el rendimiento del hotspot, como antenas IoT de alto rendimiento, cables de antena, bases de montaje y protectores contra rayos.

**Interfaz**
============

![](https://www.sensecapmx.com/wp-content/uploads/2022/07/插图-01.jpg)

**Nota:**

*   La ranura para tarjeta MicroSD está reservada para uso futuro. SenseCAP M2 no necesita tarjeta MicroSD como gateway LoRaWAN.
*   La ranura para tarjeta Nano SIM solo estará disponible en la versión 4G del hotspot. La tarjeta SIM no está incluida.
*   La interfaz Type-C está reservada para gestión avanzada de desarrolladores. No la utilices para alimentar el dispositivo.

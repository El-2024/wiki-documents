EdgeBox-RPI-200 no es solo una puerta de enlace industrial basada en Raspberry Pi/Linux y una puerta de enlace IIoT, sino también un controlador de computación perimetral todo en uno que combina múltiples propósitos industriales. Diseñado como hardware industrial de alta escalabilidad y robustez, con una amplia variedad de recursos de E/S y respaldado por el sólido ecosistema de software industrial de Raspberry Pi, es una elección ideal para soluciones de automatización inteligente e Internet Industrial de las Cosas (IIoT).

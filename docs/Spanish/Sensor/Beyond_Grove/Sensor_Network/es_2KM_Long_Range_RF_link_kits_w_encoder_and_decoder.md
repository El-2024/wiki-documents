Este es un kit de enlace RF de 433 MHz de **súper largo alcance**. Cuenta con tecnología VCO y PLL, frecuencia estable y una gran capacidad antiinterferencias. Puedes usarlo directamente en tus proyectos como transmisión de datos inalámbrica, control remoto, entre otros.

[![](https://files.seeedstudio.com/wiki/Seeed-WiKi/docs/images/300px-Get_One_Now_Banner-ragular.png)](https://www.seeedstudio.com/2km-long-range-rf-link-kits-w-encoder-and-decoder-p-321.html?cPath=139_140)

## Especificaciones
---
| Parámetro              | Valor                                     |
|------------------------|-------------------------------------------|
| Voltaje de trabajo     | Receptor (3–5 V), Transmisor (3–9 V)      |
| Corriente de trabajo   | ≤2.5 mA (5.0 VDC)                          |
| Principio de operación | Superhet (VCO, PLL)                       |
| Modulación             | OOK/ASK                                   |
| Banda de trabajo       | 433.92 MHz (servicio personalizado disponible) |
| Ancho de banda         | 1.5 MHz                                    |
| Sensibilidad           | -105 dBm (50 Ω)                            |
| Velocidad de datos     | ＜5 Kbps                                   |
| Forma de decodificación| PT2272                                     |
| Longitud de antena     | 18 cm                                      |
| Alcance de transmisión | 2 km                                       |

## Ideas de Aplicación
---
* Interruptores inalámbricos  
* Transmisión remota  
* Control inalámbrico

## Primeros Pasos
---

- Sigue las [Instrucciones](https://files.seeedstudio.com/wiki/2KM_Long_Range_RF_link_kits_w_encoder_and_decoder/res/2KM_RF.rar) para configurar el sistema de comunicación.
- Conecta la batería, el botón y las placas transmisoras.  
![](https://files.seeedstudio.com/wiki/2KM_Long_Range_RF_link_kits_w_encoder_and_decoder/img/2KM_TX.JPG)

!!! Nota
    ¡Usa una fuente de alimentación de 3–5 V! Usar 9 V dañará la placa receptora.

- Conecta la batería, los LEDs y las placas receptoras.
![](https://files.seeedstudio.com/wiki/2KM_Long_Range_RF_link_kits_w_encoder_and_decoder/img/2KM_RX.JPG)

!!! Nota
    No coloques el módulo RX y el módulo TX demasiado cerca: esto impediría su correcto funcionamiento. Asegúrate de que estén al menos a 1 metro de distancia uno del otro.

- Al presionar el botón del lado transmisor, el LED correspondiente del lado receptor se encenderá.

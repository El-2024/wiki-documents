# Grove Base Shield para NodeMCU

Grove Base Shield para NodeMCU es una placa de expansión que te ayuda a usar sensores Grove con un kit de desarrollo ESP8266 WIFI llamado NodeMCU. Puedes controlar los sensores Grove con scripts en lenguaje Lua en el firmware de NodeMCU.

[![](https://files.seeedstudio.com/wiki/Seeed-WiKi/docs/images/300px-Get_One_Now_Banner-ragular.png)](https://www.seeedstudio.com/Grove-Base-BoosterPack-p-2177.html)

## Características

---

- Compatible con todos los módulos Grove  
- 5 conectores digitales (D3-D8)  
- 1 conector analógico (A0)  
- 2 sockets I2C  
- Sin socket SPI  
- Conector UART/D9-D10  
- LED indicador de energía  

## Función de la Interfaz

---

El propósito del base shield es facilitar la conexión de pines de entrada y salida de cualquier microprocesador a los módulos Grove. Para una vista más detallada del Base Board, consulta el siguiente diagrama:

![](https://files.seeedstudio.com/wiki/Grove_Base_Shield_for_NodeMCU_V1.0/img/Base_Shield_for_NodeMCU2.jpg)

:::note

1. La alineación escalonada de los pines — por ejemplo, el socket D3 maneja D3 y D5, el socket D5 maneja D5 y D6, el siguiente D6 y D7, etc.  
2. No existe socket D4.  
3. El socket UART puede multiplexar con los puertos D9 y D10, y el socket I2C puede multiplexar con los puertos D1 y D2.  
4. No hay conflicto si usas los cuatro sockets I2C simultáneamente, porque cada dispositivo I2C tiene su propia dirección.  
5. No hay socket SPI, ya que ningún módulo Grove utiliza SPI.  

:::

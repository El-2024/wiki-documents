![](https://files.seeedstudio.com/wiki/Wio_Link_Deluxe_Plus_Kit/img/Wio_Link_Deluxe_Plus_Kit_product_view_1200_s.jpg)

Este kit es igual que el **Wio Link Deluxe Kit**, excepto que este kit contiene adicionalmente dos módulos de pantalla, que pueden agregar interfaces visuales para el usuario. Puedes usar este kit para ejecutar proyectos reales o hacer prototipos. Contiene sensores (principalmente), módulos con función de conmutación, servomotores, tira de LED y un altavoz. La parte más increíble de este kit es la placa de desarrollo Wio Link, que puede utilizarse para varios dispositivos sin necesidad de conocimientos de programación. Todas las acciones que quieras que realicen tus módulos y la operación de flasheo del código en Wio Link se pueden completar con toques en una app móvil. Además, puedes enviar instrucciones a tus dispositivos a través de internet desde cualquier lugar. Por eso, Wio Link será una utilidad perfecta para hacer que las cosas a tu alrededor sean más inteligentes para todos. ¡Ahora hagamos algo real!

[![](https://files.seeedstudio.com/wiki/common/Get_One_Now_Banner.png)](https://www.seeedstudio.com/depot/Wio-Link-Deluxe-Plus-Kit-p-2616.html)

:::caution
     La función Wio IFTTT está descontinuada (EOL). Pero la app Wio está disponible. Puedes usar la API de la app Wio para leer el estado de sensores y controlar actuadores.
:::

--------

- Paquete completo con todos los módulos funcionales generales necesarios  
- Interfaz con puerto Grove, funciones más prácticas y menos trabajo  
- Completamente fácil para hacer que las cosas a tu alrededor sean más inteligentes y agregar diversión a la vida diaria  
- Rentable

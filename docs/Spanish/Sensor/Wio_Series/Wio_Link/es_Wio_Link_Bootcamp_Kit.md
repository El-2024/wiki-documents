Si no tienes habilidades de programación y quieres construir gadgets geniales para tu vida, este kit será un excelente inicio. Contiene un relé, un grabador, una tira de LEDs, un buzzer y un servo motor que realizarán acciones básicas necesarias en la vida diaria. La parte más increíble de este kit es la placa de desarrollo Wio Link, que puede utilizarse para varios dispositivos: sin necesidad de conocimientos de programación. Todas las acciones que quieras que realicen tus módulos y la operación de flasheo del código en Wio Link se pueden completar con toques en una app móvil. Además, puedes enviar instrucciones a tus dispositivos a través de Internet desde cualquier lugar. Por eso, Wio Link será una utilidad perfecta para hacer las cosas a tu alrededor más inteligentes para todos. Ahora, comencemos a construir cosas útiles.

[![](https://files.seeedstudio.com/wiki/common/Get_One_Now_Banner.png)](https://www.seeedstudio.com/depot/Wio-Link-Bootcamp-Kit-p-2613.html?cPath=19_20)

:::caution
     La función Wio IFTTT está descontinuada (EOL). Pero la app Wio está disponible. Puedes usar la API de la app Wio para leer el estado de sensores y controlar actuadores.
:::

Características
--------

- Incluye módulos funcionales comunes frecuentemente usados para monitoreo ambiental  
- Interfaz con puerto Grove, funciones más prácticas y menos trabajo  
- Completamente fácil para hacer que las cosas a tu alrededor sean más inteligentes y divertidas  
- Rentable  
- Peso: 285 g

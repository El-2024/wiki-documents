Siempre estamos estrechamente conectados con nuestro entorno. Por eso es importante obtener algunos detalles del ambiente. Este kit está pensado para realizar esta tarea perfectamente, al menos a un nivel básico. En este kit puedes encontrar módulos funcionales comunes para recopilar datos, como humedad o nivel de luz interior, del ambiente de nuestra casa. La parte más sorprendente de este kit es la placa de desarrollo Wio Link, que puede usarse para varios propósitos sin necesidad de habilidades de programación. Todas las acciones que quieres que tus módulos realicen y la operación de flasheo del código en Wio Link pueden completarse con toques en una app móvil. Y puedes enviar instrucciones a tus dispositivos por Internet desde cualquier lugar. Por eso, Wio Link será una utilidad perfecta para hacer que las cosas a tu alrededor sean más inteligentes para todos. Así que vamos a entender mejor las condiciones en las que estás.

[![](https://files.seeedstudio.com/wiki/common/Get_One_Now_Banner.png)](https://www.seeedstudio.com/depot/Wio-Link-Environment-Kit-p-2612.html)

:::caution
     La función Wio IFTTT está descontinuada (EOL). Pero la app Wio está disponible. Puedes usar la API de la app Wio para leer el estado de sensores y controlar actuadores.
:::

Características
--------

- Incluye módulos funcionales comunes para monitoreo ambiental  
- Interfaz con puerto Grove, función más práctica y menos trabajo  
- Rentable  
- Peso: 154 g

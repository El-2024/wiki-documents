# SenseCAP Watcherは何をするのか

<div style={{textAlign:'center'}}><img src="https://files.seeedstudio.com/wiki/watcher_getting_started/50.jpg" style={{width:800, height:'auto'}}/></div>

インテリジェント空間に対する認識を再定義する旅に出る準備をしてください。画期的な物理AIエージェントであるWatcherは、あなたの環境にシームレスに統合され、周囲の世界との相互作用や体験の仕方を変革します。顔認識、物体検出、マルチシナリオセンシングなどの先進技術の力を活用することで、Watcherはより賢く、より直感的で、高度に適応可能な空間を創造します。

このwikiでは、物理AIエージェントとしてのWatcherの膨大な可能性を示す3つの変革的なアプリケーションを探求します。これらのアプリケーションは、Watcherの能力を実証するだけでなく、あなた自身の革新的なソリューションを構想し開発するためのインスピレーションを与えます。

- **Watcherアプリケーション #1: 訪問者への道案内**: Watcherがインテリジェントガイドとして機能することで、訪問者体験をどのように革命化するかを目撃してください。顔認識機能により、Watcherは訪問者を識別し、個人化された道案内とサポートを提供し、あなたの空間の効率性とアクセシビリティを向上させます。

- **Watcherアプリケーション #2: フラワーショップのバーチャルレジ係**: Watcherが無人フラワーショップでバーチャルレジ係として機能することで、小売業界をどのように変革できるかを発見してください。顧客が購入準備ができたことを検出することで、WatcherはシームレスなセルフチェックアウトのためのQRコードを迅速に表示し、店舗運営を最適化しながら、スムーズで安全な取引プロセスを確保します。

- **Watcherアプリケーション #3: オールインワンセンサー - 転倒検出、ペット監視、紛失物アラート**: オールインワンセンサーソリューションとしてのWatcherの汎用性を解き放ちます。転倒検出からペット監視、紛失物の警告まで、Watcherは様々なロングテールシナリオに適応し、より安全で、より安心で、高度に応答性の高い環境を創造します。

これらのアプリケーションを深く掘り下げることで、物理AIエージェントとしてのWatcherが、私たちが住む空間をどのように再形成し最適化できるかについて、より深い理解を得ることができます。あなたがイノベーター、ビジネスオーナー、または単に技術の未来に情熱を持つ人であっても、このwikiはWatcherの能力を活用し、より賢く、より適応可能な空間を創造するために必要な知識とインスピレーションを提供します。

Watcherでインテリジェント環境の境界を再定義するこの変革的な旅に参加してください！


## デモ 1. 訪問者への道案内

このアプリケーションでは、特に別荘での配達員の案内という文脈で、訪問者体験を向上させるためにWatcherをどのように活用できるかを探求します。2台のWatcherデバイスを戦略的に配置し、特定のコマンドで設定することで、訪問者を案内し、正確な荷物配達を確保するシームレスで効率的なプロセスを作成できます。

### ステップ 1. 最初のWatcherの配置

最初のWatcherは、配達員が容易に視認できる場所の別荘の入り口に設置する必要があります。このWatcherは最初の接触点として機能し、荷物を持った訪問者の存在を認識し、配達物をどこに置くべきかについて明確な指示を提供します。

### ステップ 2. 2番目のWatcherの配置

2番目のWatcherは、別荘内の指定された荷物保管エリアに配置する必要があります。このWatcherは、荷物の正確な配置を確認し、配達員のサービスに対して感謝の意を表すことを担当します。

### ステップ 3. 最初のWatcherのコマンド設定

SenseCraft APPを使用して、最初のWatcherに以下のコマンドを発行します：

**もし誰かが荷物を持っていたら、「こんにちは、配達員さん、荷物は右側のテーブルまで、まっすぐ3メートル先です、ありがとうございます」と言ってください。**

このコマンドにより、Watcherは配達員に明確で簡潔な指示を提供し、適切な荷物保管エリアへと案内することが確保されます。

<div style={{textAlign:'center'}}><img src="https://files.seeedstudio.com/wiki/watcher_getting_started/51.png" style={{width:600, height:'auto'}}/></div>

### ステップ 4. 2番目のWatcherのコマンド設定

SenseCraft APPを使用して、2番目のWatcherに以下のコマンドを発行します：

**もし誰かがテーブルに荷物を置いたら、「ありがとうございます」と言ってください。**

このコマンドにより、2番目のWatcherは荷物の配置成功を確認し、配達員の努力に対して感謝を表現できます。

<div style={{textAlign:'center'}}><img src="https://files.seeedstudio.com/wiki/watcher_getting_started/52.png" style={{width:600, height:'auto'}}/></div>

Watcherアプリケーション #1は、慎重に作成されたコマンドと組み合わせたWatcherデバイスの戦略的展開が、別荘での訪問者体験を大幅に向上させ、荷物配達プロセスを合理化できることを実証しています。

このアプリケーションは、物理AIエージェントとしてのWatcherの汎用性と可能性の証として機能し、インテリジェント空間内での私たちの日常生活と相互作用の様々な側面を変革し最適化する能力を示しています。

## デモ 2. フラワーショップのバーチャルレジ係

このアプリケーションでは、Watcherがバーチャルレジ係として機能することで、フラワーショップでのショッピング体験をどのように革新できるかを探ります。Watcherの物体検出機能とSenseCraft APPを活用することで、シームレスで効率的なセルフチェックアウトプロセスを作成し、顧客体験を向上させ、店舗運営を合理化できます。

### ステップ 1. 決済QRコードの生成

PayPalまたは他の好みの決済プロバイダーなどのプラットフォームを使用して決済QRコードを生成することから始めます。このQRコードは、セルフチェックアウトプロセスの重要な要素として機能し、顧客が花の購入に対して簡単に支払いを行えるようにします。

### ステップ 2. SenseCraft APPを介してWatcherにQRコードをアップロード

SenseCraft APPにアクセスし、**Animation**セクションに移動します。**Watching Space**を見つけて、既存のアニメーションを生成した決済QRコードに置き換えます。このステップにより、WatcherがトリガーされたときにQRコードを表示し、スムーズなセルフチェックアウトプロセスを可能にします。

### ステップ 3. 三脚にWatcherを取り付け

最適なパフォーマンスと安定性を確保するため、1/4ネジ三脚ブラケットを使用してWatcherを三脚に取り付けます。このセットアップにより、Watcherの位置と角度を簡単に調整でき、チェックアウトエリアの明確な視認性を確保できます。

### ステップ 4. チェックアウトカウンターでのWatcherの配置

三脚に取り付けたWatcherをチェックアウトカウンターに配置し、顧客が支払いのために近づくことが予想されるエリアに向けます。Watcherがカウンターを明確に見渡せ、花を持った顧客を簡単に検出できることを確認してください。

### ステップ 5. SenseCraft APPを通じたWatcherのタスク設定

SenseCraft APPを使用して、Watcherに以下のコマンドを発行します：

**花を持ってあなたに向かっている人を検出したら、「QRコードをスキャンして購入を完了してください」と言ってください。**

このコマンドは、Watcherに花を持った顧客を監視し、アップロードした決済QRコードを表示し、コードをスキャンしてセルフチェックアウトプロセスを開始するよう促すよう指示します。

<div style={{textAlign:'center'}}><img src="https://files.seeedstudio.com/wiki/watcher_getting_started/53.png" style={{width:600, height:'auto'}}/></div>

### ステップ 6. セルフチェックアウトプロセスの実装

花を持った顧客を検出すると、Watcherは自動的に決済QRコードを表示し、事前設定された音声プロンプトをトリガーして、顧客に支払いのためのコードスキャンを案内します。

顧客がQRコードをスキャンすると、モバイル決済インターフェースに誘導され、好みの支払い方法を使用して取引を完了できます。


Watcherアプリケーション #2は、フラワーショップ環境でのバーチャルレジ係としてのWatcherの変革的な可能性を示しています。Watcherの物体検出機能とSenseCraft APPを組み合わせることで、フラワーショップのオーナーは、シームレスで効率的なセルフチェックアウト体験を作成し、顧客満足度を向上させ、店舗運営を最適化できます。

このアプリケーションは、Watcherの汎用性と適応性を実証し、さまざまな小売環境を革新する可能性を強調しています。物理的なAIエージェントとして、Watcherは異なるビジネスの特定のニーズに合わせてカスタマイズおよび設定でき、イノベーションを推進し、全体的なショッピング体験を向上させます。

## デモ 3. あなたのオールインワンセンサー -- 転倒検知、ペット監視、忘れ物アラート

このアプリケーションでは、幅広いロングテール・シナリオに対応できるオールインワンセンサーソリューションとしてのWatcherの汎用性と適応性を探求します。複数のWatcherデバイスを異なる場所に戦略的に配置し、特定の状況に対処するよう設定することで、様々な環境における安全性、セキュリティ、利便性を向上させるWatcherの可能性を実証できます。

### シナリオ 1: 会議室での忘れ物アラート

ステップ 1: 最初のWatcherを360°回転ブラケットを使用して会議室の壁に取り付け、部屋全体を最適にカバーできるようにします。

ステップ 2: SenseCraft APPを使用して、Watcherに以下のコマンドを発行します：

**会議室が空で、テーブルにコンピューターがあることを検知したら、アプリに通知を送信してください。**

このタスクにより、Watcherは会議室を監視し、コンピューターなどの貴重品が放置されている場合にアプリを通じてユーザーにアラートを送信できます。

<div style={{textAlign:'center'}}><img src="https://files.seeedstudio.com/wiki/watcher_getting_started/54.png" style={{width:600, height:'auto'}}/></div>

### シナリオ 2: ペット監視と抑制

ステップ 1: 1/4ネジ三脚ブラケットを使用して2台目のWatcherを三脚に設置し、花瓶の近くや猫が損害を与えやすい場所に配置します。

ステップ 2: SenseCraft APPを使用して以下のタスクを設定します：

**猫を検知したら、「危険です、離れてください」という音声メッセージを再生してください**

猫を近づけないよう抑制します。

<div style={{textAlign:'center'}}><img src="https://files.seeedstudio.com/wiki/watcher_getting_started/55.png" style={{width:600, height:'auto'}}/></div>

ステップ 3: Watcherの音量を最大に調整し、猫を効果的に驚かせて制限区域への侵入を阻止します。

### シナリオ 3: 両親の家での転倒検知

ステップ 1: 3台目のWatcherを両親の家の壁に設置し、主要な生活エリアを明確に見渡せるようにします。

ステップ 2: SenseCraft APPを通じて以下のタスクを割り当てます：

**地面に横たわっている人を検知したら、すぐに私に通知してください。**

この設定により、Watcherは潜在的な転倒や事故を監視し、必要な時に迅速な対応と支援を確保するため、ユーザーに即座にアラートを送信できます。

<div style={{textAlign:'center'}}><img src="https://files.seeedstudio.com/wiki/watcher_getting_started/56.png" style={{width:600, height:'auto'}}/></div>

Watcherアプリケーション #3は、オールインワンセンサーソリューションとしてのWatcherの驚くべき汎用性と適応性を実証しています。転倒検知、ペット監視、忘れ物アラートなど、幅広いロングテール・シナリオに対処することで、Watcherは様々な環境における安全性、セキュリティ、利便性を革新する可能性を示しています。

このアプリケーションは、特定のニーズに合わせてカスタマイズおよび設定できるWatcherの能力を強調し、個人、家族、企業にとって貴重なツールとなることを示しています。物理的AIエージェントとして、Watcherは異なる環境にシームレスに統合され、プロアクティブな監視、リアルタイムアラート、そしてユーザーに安心感を提供します。

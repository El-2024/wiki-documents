愛好家から産業従事者まで、さまざまなユーザーのニーズに対応する製品群を提供するSenseCAP製品ラインは、私たちの周りの世界をデジタル化するための包括的なソリューションスイートを提供します。

センサーからゲートウェイまで、ユーザーは環境のさまざまな側面を監視・制御するための独自のソリューションを構築・カスタマイズできます。製品ラインには、大規模展開が可能な産業グレードのデバイスも含まれており、高度な機能と堅牢な性能を提供します。

さらに、SenseCAP製品ラインは、ソフトウェア、クラウドサービス、APIの配列も提供し、ユーザーがリアルタイムでデータを収集、保存、分析できるようにします。ハードウェア、ソフトウェア、クラウドサービスの組み合わせにより、ユーザーは環境の変化に迅速に対応し、データ駆動型の意思決定を行うことができます。

## Wild IoT

Wild IoTとは、自然環境にセンサーネットワークを展開して、さまざまな環境データをリアルタイムで監視・収集する技術を指します。このデータには、温度、湿度、気圧、光強度、土壌水分などが含まれ、科学研究、生態保全、資源管理に役立ちます。

### なぜ産業用センサーが必要なのか？

1. **耐久性と信頼性**: 屋外環境は複雑で変化に富んでいます。産業用センサーは、高温、湿度、腐食に対する耐性を持ち、過酷な条件下で長期間安定して動作できます。
2. **高精度データ収集**: 科学研究と環境監視には高精度データが必要です。産業用センサーの高精度と安定性がこれらの要求を満たします。
3. **低消費電力と長いバッテリー寿命**: 野外では電源供給が困難な場合があります。産業用センサーは通常、低消費電力デバイスとして設計されており、長時間の動作を可能にし、メンテナンスとバッテリー交換の頻度を減らします。

### どのように動作するのか？

1. **データ収集**: 産業用センサーを屋外環境に設置し、温度、湿度、気圧などのリアルタイムデータを収集します。
2. **データ送信**: 収集されたデータは、LoRaWANなどの低消費電力広域ネットワーク（LPWAN）技術を使用してゲートウェイに送信されます。
3. **データ処理と保存**: ゲートウェイはデータをクラウドプラットフォームに送信し、処理、保存、分析を行います。
4. **データ可視化と応用**: ユーザーは、クラウドプラットフォーム上のアプリケーションやダッシュボードを通じてリアルタイムでデータを表示し、分析と意思決定に役立てることができます。

### SenseCAP Mate APP

[SenseCAP Mate APP](https://wiki.seeedstudio.com/Cloud_Chain/SenseCAP_Mate_APP/SenseCAP_APP) は、データ可視化とリモートデバイス管理のための強力なアプリです。

- クリーンなユーザーインターフェースでデータを取得し表示します
- 時間間隔を設定：バッテリー効率を向上させます
- 閾値を設定：アクションを取る必要があるデータに焦点を当てます - 近日公開
- データが安全範囲外の場合にアラームを送信 - 近日公開

### SenseCAP AI

[SenseCAP AI](https://wiki.seeedstudio.com/How_to_Use_SenseCAP_AI_on_SenseCAP_Portal_and_SenseCAP_Mate_APP/) は、センサーデータを最大限に活用し、センサーデータの潜在能力を完全に引き出すのに役立つAI駆動ソリューションです。SenseCAP センサーを SenseCAP プラットフォームに接続することで、温度、湿度、光、空気品質などの環境要因に関するデータを簡単に収集・分析できます。同時に、私たちの強力なAIアルゴリズムがこのデータを使用して、運用の最適化、コスト削減、効率向上に役立つ実用的な洞察と推奨事項を提供します。

<p style={{textAlign: 'center'}}><img src="https://files.seeedstudio.com/products/102991154/wiki%20images/portal.png" alt="pir" width={800} height="auto" /></p>

空気品質の監視、作物成長の最適化、効率の向上など、どのような目的であっても、SenseCAP AI には成功に必要なツールがあります。SenseCAP AI は現在 [SenseCAP Portal](https://sensecap.seeed.cc/portal/#/login) と [SenseCAP Mate App](https://sensecap-mate-download.seeed.cc/) でリリースされており、ユーザーはいつでも使用できます。使いやすいプラットフォームとアプリにより、センサーを素早く接続し、わずか数分で貴重な洞察を受け取ることができます。

### SenseCAP API

SenseCAP Application Programming Interface を使用することで、特定の要求に応じて独自の SenseCAP デバイスとデータを管理できます。柔軟に適用できる3種類のAPIメソッドがあります：

- HTTP API を使用すると、LoRa および NB-IoT デバイスを管理し、RAW データまたは履歴データを取得できます。
- MQTT API を使用すると、MQTT プロトコルを通じてセンサーのリアルタイム測定データを購読できます。
- WebSocket API を使用すると、Websocket プロトコルを通じてセンサーのリアルタイム測定データを取得できます。

SenseCAP API のガイドとリファレンスについて詳しく知るには、[SenseCAP Document Center – API](https://sensecap-docs.seeed.cc/introduction.html) をご覧ください。

Grove - I2C Hub (6 Port) は、Grove Base Shield に複数の I2C デバイスを接続するための拡張 Grove モジュールです。[Universal 4 Pin to X2 4 Pin ケーブル](https://www.seeedstudio.com/depot/universal-4-pin-to-x2-4-pin-cable-5-pcs-pack-p-847.html?cPath=178_179)と組み合わせて使用することで、最大 5 つの I2C デバイスを接続でき、ほとんどの開発目的をカバーすることができます。Grove - I2C Hub (6 Port) は Grove - I2C Hub よりも 2 つ多いピンを持っており、より多様なニーズに対応します。

<p style={{textAlign: 'center'}}><a href="https://www.seeedstudio.com/Grove-I2C-Hub-6-Port-p-4349.html" target="_blank"><img src="https://files.seeedstudio.com/wiki/wiki_english/docs/images/get_one_now_small.png" width={200} height={38} border={0} /></a></p>

## 特徴

- 連結可能
- [Grove - I2C Hub](https://www.seeedstudio.com/Grove-I2C-Hub.html) よりも 2 つ多いコネクタ
- 経済的で実用的

## 仕様
  
| パラメータ | 値/範囲 |
|---|---|
| 動作電圧 | 3.3v/5v |
| 動作温度 | -25℃ ～ +85℃ |
| サイズ | 40mm x 20mm x 15mm |

## 応用アイデア

- Grove プラットフォーム上で、Grove I2C Hub よりも 2 つ多いコネクタを使用して、より多くの I2C デバイスを使用する。

## 使用方法

Grove I2C Hub と同様に、同じ I2C ソケットを同時に使用しても競合しません。これは、各 I2C デバイスが独自のアドレスを持っているためです。ハードウェアの設置は以下の図の通りです。

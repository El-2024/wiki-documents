# アプリ設定

設定では、`SenseCAP Hotspot App`でPINを有効にしたり、サポートドキュメントを取得したり、ヘルプセンターでサポートチケットを開いたり、heliumウォレットを切り替えたり、heliumウォレットを終了したりできます。

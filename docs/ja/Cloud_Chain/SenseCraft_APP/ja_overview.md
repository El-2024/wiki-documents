# 紹介と使用方法

SenseCraft APPは、データ可視化とデバイス管理のための強力なAPPです。

## アプリのハイライト

- 時間の節約 - スキャンアンドプレイ体験を提供し、ユーザーが4ステップで全体の設定を完了できます
- 技術的背景が少ない、または全くない人でもセンサーデータを簡単に始められます
- クリーンなユーザーインターフェースでデータを取得・表示
- 時間間隔の設定：バッテリー効率を向上
- 閾値の設定：アクションを取る必要があるデータに焦点を当てる - 近日公開
- データが安全範囲外の場合にアラームを送信 - 近日公開

## ダウンロード

SenseCraft AppはiOSとAndroidの両バージョンで利用可能です。

## アカウント

SenseCraftはデバイス設定とリモート管理をサポートしています。SenseCAP Portalプラットフォームとその他の機能を使用するには、アカウントを登録してください。

:::tip Note
サーバーロケーションは`Global`を選択してください。<a href="http://sensecap.seeed.cc">SenseCAP Portal</a>からもアカウントを作成できます
:::

## イベント

通知を受け取るためにイベントを追加します。
<p style={{textAlign: 'center'}}><img src="https://files.seeedstudio.com/wiki/SenseCAP/introduction/add-event.PNG" alt="pir" width={300} height="auto" /></p>

1. 追加アイコンまたはイベント追加ボタンをクリックしてイベントアラートを作成し、イベント追加ページの条件に条件オプションを追加し、追加ボタンをクリックしてデバイスを選択します。

<div style={{textAlign:'center'}}><img src="https://files.seeedstudio.com/wiki/sensecap_mate_app/mate_app_2.png" style={{width:1000, height:'auto'}}/></div>

2. デバイスを選択し、測定タイプを選択し、アラーム条件を設定し、より大きいまたはより小さいなどの条件を選択してプログレスバーをドラッグして値を設定し、次のステップをクリックします。

<div style={{textAlign:'center'}}><img src="https://files.seeedstudio.com/wiki/sensecap_mate_app/mate_app_3.png" style={{width:1000, height:'auto'}}/></div>

3. イベント追加ページの正常復帰アクションでデバイスが正常に戻った場合、通知を送信するかどうかを選択し、保存をクリックし、イベント名を入力し、送信をクリックしてイベントアラームの追加が成功し、イベントリストに戻ります。

<div style={{textAlign:'center'}}><img src="https://files.seeedstudio.com/wiki/sensecap_mate_app/mate_app_4.png" style={{width:1000, height:'auto'}}/></div>

4. デバイスページで、メッセージセンターをクリックしてアラームメッセージを表示し、デバイス警告デバイスがトリガーしたアラームプッシュメッセージ、システム通知システムメッセージを表示します。

<div style={{textAlign:'center'}}><img src="https://files.seeedstudio.com/wiki/sensecap_mate_app/mate_app_5.png" style={{width:1000, height:'auto'}}/></div>

5. アラームメッセージをクリックし、デバイスが条件項目をトリガーし、アラームメッセージをプッシュし、クリックしてアラーム詳細を表示します。アラームリストに戻るとステータスが既読に変わり、編集ボタンをクリックしてメッセージを選択し、既読、削除などの操作ができます。

<div style={{textAlign:'center'}}><img src="https://files.seeedstudio.com/wiki/sensecap_mate_app/mate_app_6.png" style={{width:1000, height:'auto'}}/></div>

6. システムメッセージを切り替え、クリックしてシステムプッシュメッセージの詳細を表示します。

<div style={{textAlign:'center'}}><img src="https://files.seeedstudio.com/wiki/sensecap_mate_app/mate_app_7.png" style={{width:1000, height:'auto'}}/></div>

 <!-- Q4: How do I fix my device when I faced the "Read Only Root filesystem"? -->

"Read Only Root filesystem"状態のデバイスに対しては、`e2fsck`を使用してシステムをチェックし、修復することができます。ターミナルウィンドウを開いて、以下のコマンドを実行してください：

```
e2fsck /dev/mmcblk0p2
```

<!-- ### Q1: What is the password of default user "recomputer"? -->

システムには事前設定されたユーザー名とパスワードがあります。ログイン時にはデフォルトユーザー名を「**recomputer**」、デフォルトパスワードを「**12345678**」に設定してください。異なる認証情報を設定すると、問題が発生する可能性があります。この問題は現在解決中です。

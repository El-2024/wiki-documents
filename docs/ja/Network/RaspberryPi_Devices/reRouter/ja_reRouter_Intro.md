この Mini Router は、デュアル Gigabit Ethernet ポートとデュアル USB 3.0 ポートを備えており、ソフトルーターアプリケーションに適している一方で、ハードウェアを最小限に抑えています。Raspberry Pi Compute Module 4（4GB RAM、32GB eMMC）を事前にインストールしており、迅速な起動時間とスムーズな全体的な使用体験を提供します。

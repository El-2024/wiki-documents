**SenseCAP M2 Data Only とは**

SenseCAP M2 Data Only は、Helium LongFi ネットワークに接続された LoRaWAN ゲートウェイであり、長距離 / LoRaWAN デバイス向けに数マイルの無線ネットワークカバレッジとデータ伝送能力を提供することができます。

LoRaWAN ゲートウェイとして、長距離 / LoRaWAN デバイス向けに数マイルの無線ネットワークカバレッジとデータ伝送能力を提供することができます。

Helium LongFi ネットワークをサポートするゲートウェイとして、ローカル LoRaWAN ネットワークを構築し、低コストでさまざまな LoRaWAN アプリケーションを開発することができると同時に、People's Network の一部としての利点を享受することができます。

SenseCAP M2 は現在、FCC および CE 認証を取得した US915 および EU868 バージョンを提供しています。近い将来、さまざまな国や地域向けの周波数帯バージョンと対応する認証がさらに利用可能になる予定です。

**特徴**
============

![](https://www.sensecapmx.com/wp-content/uploads/2022/07/sensecap-m2-DO-features.png)

**· 広範囲のカバレッジと強力な信号**

3dBi 高性能アンテナを搭載した SenseCAP M2 は、数マイルの LoRaWAN カバレッジと強力な信号を提供します。また、異なる環境に基づいて使用シナリオに合わせてアンテナをアップグレードすることができます。

**· 優れた安定した性能**

SenseCAP M2 は成熟したハードウェアソリューション MT7628 と Semtech SX1302/SX1303 ベースバンド長距離チップによって駆動されます。内蔵 BLE を搭載し、Wi-Fi とイーサネットのインターネット接続の両方をサポートします。イーサネットでゲートウェイを電源供給する必要があるユーザー向けに、PoE 機能も M2 に追加されています。

**· プロフェッショナルな管理ツールとクラウドサービス**

ユーザーは SenseCAP Hotspot App を使用して簡単な手順でホットスポットをセットアップできます。SenseCAP Hotspot Dashboard と SenseCAP Local Console も開発されており、ユーザーが効率的かつ簡単にホットスポットを監視および管理することができます。

**· 効率的でタイムリーなオンラインサポート**

これらのプロフェッショナルな管理ツールとクラウドサービスに加えて、SenseCAP は顧客から技術サポートで高く評価されています。技術的な問題に直面した際には、効率的なオンライン技術サポートサービスが常にここにあり、タイムリーにサポートします。

**· 安全で信頼性の高い**  
大量のデータ伝送を処理するゲートウェイにとって、安全性は常に重要です。SenseCAP M2 Data Only は、People's Network への高いセキュリティ認証と信頼性の高い接続を常に保証します。

**· 簡単な手順でセットアップ**

SenseCAP Hotspot App を使用して簡単な手順で準備完了

**· 豊富なアクセサリー**

SenseCAP デバイス自体の強力な機能に加えて、SenseCAP はユーザーがホットスポットの最大性能を得るための豊富なアクセサリー選択肢を提供しています。例えば、高性能 IoT アンテナ、アンテナケーブル、マウントパッド、雷防止器などのベストセラーがあります。

**インターフェース**
=============

![](https://www.sensecapmx.com/wp-content/uploads/2022/07/插图-01.jpg)

**注意:**

*   MicroSD カードスロットは将来の使用のために予備されています。SenseCAP M2 を LoRaWAN ゲートウェイとして使用する場合、MicroSD カードは必要ありません。
*   Nano SIM カードスロットは 4G バージョンのホットスポットでのみ利用可能です。SIM カードはデバイスに含まれていません。
*   Type-C インターフェースは高度な開発者管理のために予備されています。デバイスの電源供給には使用しないでください。

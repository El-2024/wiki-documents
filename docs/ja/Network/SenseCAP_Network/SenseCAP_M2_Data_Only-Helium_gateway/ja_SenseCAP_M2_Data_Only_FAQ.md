FAQ
===

**SenseCAP M2はライトゲートウェイですか？**
===================================

今回リリースしたSenseCAP M2 Data Only LoRaWANゲートウェイは、HeliumネットワークのData-Onlyタイプのゲートウェイです。

* * *

**ライトホットスポットもリリースされる予定ですか？**
=========================================

ライトゲートウェイはHeliumから正式にリリースされていません。ライトゲートウェイの開発アップデートについては、Heliumの発表をフォローしてください。これは私たちのロードマップに含まれています。今後のアップデートを受け取るために、私たちのソーシャルメディアチャンネルをフォローしていることを確認してください。

* * *

**SenseCAPはいつHelium Light Gatewayをリリースしますか？**
========================================================

今回リリースしたSenseCAP M2 Data Only LoRaWANゲートウェイは、HeliumネットワークのData-Onlyタイプのゲートウェイです。ライトゲートウェイはHeliumから正式にリリースされていません。ライトゲートウェイの開発アップデートについては、Heliumの発表をフォローしてください。ライトゲートウェイのアップデートがある際は、私たちの公式チャンネルでも公開します。お楽しみに！

* * *

**SenseCAP M1とSenseCAP M2 Data-Onlyの違いは何ですか？**
=========================================================================

SenseCAP M1は、Heliumネットワークをサポートし、PoC（カバレッジ証明）とデータ送信から報酬を得ることができるLoRaWANゲートウェイです。SenseCAP M2 Data Onlyは、Helium LongFiネットワークをサポートし、データ送信から報酬を得るLoRaWANゲートウェイです。データオンリー版ではPoC報酬はサポートされていません。

* * *

**SenseCAP M2 Data OnlyはSenseCAP M1のようにPoCから報酬を得ることをサポートしていますか？**
===========================================================================

いいえ。SenseCAP M2 Data OnlyはLoRaWANデータを転送し、報酬を得ることができるLoRaWANゲートウェイです。PoC報酬はサポートされていません。

* * *

**SenseCAPは私のHeliumウォレットのパスワードをリセットできますか？**
=============================================================

絶対にできません。12の単語は、紛失した場合、誰も復元することはできません。**必ず**覚えておくか、書き留めてください。「**12の単語を失えば、すべてを失う**」

* * *

**SenseCAPは工場出荷時リセットを通じて私のホットスポットのバインドを解除できますか？**
=======================================================================

ウォレットにログインせずに、ウォレットに登録されたホットスポットを「バインド解除」するための工場出荷時リセットは利用できません。これはブロックチェーンアプリケーションの最も基本的なセキュリティ保証です。

ホットスポットを他の人に譲渡したい場合は、ログインしてHelium Hotspot AppとSenseCAP Hotspot Appで行う必要があります。

* * *

**ホットスポットの報酬とステータスはどこで確認できますか？**
======================================================

ホットスポットの報酬とブロックチェーンアクティビティは、SenseCAP Hotspot AppまたはHelium Explorer（**https://explorer.helium.com**）で確認してください。ホットスポットのステータスは、SenseCAP Dashboard（**[https://status.sensecapmx.cloud/#/dashboard](https://status.sensecapmx.cloud/#/dashboard/)**）で確認できます。

* * *

**カバレッジを向上させるためにアンテナをアップグレードできますか？**
======================================================

RF機器を使用する際は、現地の法律と規制に従ってください。ハードウェアの観点から、同じ周波数のLoRaアンテナを使用できます。RP-SMAオスコネクタ付きのアンテナまたはケーブルを選択する必要があります。アンテナを変更する際は、アプリでアンテナ情報を更新してください。

* * *

**SenseCAP M2はPoEをサポートしていますか？**
=================================

SenseCAP M2はPoE（Power on Ethernet）をサポートしています。モデム/ルーターがPoEをサポートしていない場合は、追加のPoEスイッチが必要です。PoEスイッチは40V-57V DC電源入力で、12-2A出力をサポートする必要があります。

* * *

**ホットスポットのシリアル番号（S/N）はどこで確認できますか？**
============================================================

デバイスのS/Nは、デバイス下部のラベルとパッケージのラベルから確認できます。新しいホットスポットを追加する際の今後の使用のために、S/Nと対応するHeliumデバイス名をメモしておくことをお勧めします。

* * *

**SenseCAP M2のアンテナゲインは何ですか？**
============================================

3dBiです。3dBi高性能アンテナにより、SenseCAP M2は数マイルのLoRaWANカバレッジと強力な信号を提供できます。また、異なる環境に基づいて使用シナリオに合わせてアンテナをアップグレードすることも可能です。

* * *

**SenseCAP M2にはPoE機能がありますか？**
=======================================

はい。SenseCAP M2にはPoE機能があります。ルーターまたはモデムにPoEがない場合は、セットアップに追加のPoEインジェクターが必要です。

* * *

**SenseCAP M2は何種類のバージョンが利用可能ですか？**
===================================================

SenseCAP M2は現在、FCCおよびCE認証を取得したUS915およびEU868バージョンを提供しています。そして、異なる国や地域向けに、より多くの周波数帯バージョンと対応する認証が間もなく利用可能になります。

* * *

**SenseCAP M2は屋内用または屋外用ゲートウェイですか？**
================================================

SenseCAP M2には屋内用と屋外用の両方のバージョンがあり、SenseCAP屋外エンクロージャーを選択して、屋内用のSenseCAP M1およびM2を屋外用バージョンにアップグレードできます。

SenseCAPは、市場の一般的なホットスポットに適合する防水屋外エンクロージャーを開発しました。これにより、ユーザーがゲートウェイを屋外に設置する可能性がさらに広がり、より多様なLoRaWANシナリオが現実になります。

* * *

**AS923、AU915、IN865、その他の周波数はありますか？**
=============================================================

はい。SenseCAP M2は、米国およびEU市場向けにFCCおよびCE認証を取得したUS915およびEU868バージョンがあります。そして、異なる国や地域向けに、より多くの周波数帯バージョンと対応する認証が利用可能になります。

* * *

**すべてのSenseCAP LoRaWANゲートウェイはHeliumホットスポットですか？**
======================================================

SenseCAP MXシリーズのSenseCAP M1およびM2は、Heliumネットワークをサポートしています。SenseCAPをGoogleで検索すると、SenseCAPという名前の他のゲートウェイも見つかります。SenseCAPブランドのゲートウェイがすべてHeliumホットスポットではないことにご注意ください。

* * *

**SenseCAP M2はどのタイプのSDカードを使用しますか？**
==============================================

SenseCAP M2にはSDカードは必要ありません。

Wio Tracker 1110 Dev ボードはデフォルトでオンボード LoRa アンテナを使用していますが、より良い信号を得るために外部アンテナが必要な場合があります。この章では、外部 LoRa アンテナを接続するためにアンテナパスを変更する方法をユーザーに説明します。

- `R3` を取り外します。
- `R4` を 0R に変更します。

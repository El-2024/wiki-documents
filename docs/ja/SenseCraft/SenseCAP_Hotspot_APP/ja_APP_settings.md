# APP設定

設定では、SenseCAPアプリを使用してPINを有効にしたり、サポートドキュメントを取得したり、ヘルプセンターでサポートチケットを開いたり、Heliumウォレットを切り替えたり、Heliumウォレットから退出したりすることができます。

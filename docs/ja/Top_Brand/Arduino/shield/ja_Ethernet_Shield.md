Arduino Ethernet Shieldは、Arduinoボードをインターネットに接続することを可能にします。これはWiznet W5100/W5200イーサネットチップを基盤としており、TCPおよびUDPの両方をサポートするネットワーク（IP）スタックを提供します。Ethernetライブラリを使用して、RJ45イーサネットジャックを介してインターネットに接続するスケッチを作成できます。

|![](https://files.seeedstudio.com/wiki/Ethernet_Shield/img/Seeeduino_ethernet-2.jpg)|![](https://files.seeedstudio.com/wiki/Ethernet_Shield/img/Ethernet_01.jpg)|![](https://files.seeedstudio.com/wiki/Ethernet_Shield/img/W5200_Ethernet_Shield.jpg)|
|---|---|---|
|[Seeeduino Ethernet](https://seeeddoc.github.io/Seeeduino_Ethernet/) |[Ethernet Shield V1.0](https://seeeddoc.github.io/Ethernet_Shield_V1.0/) |[Ethernet Shield V2.0](https://seeeddoc.github.io/Ethernet_Shield_V2.0/)|

以下は、各バージョンのEthernet Shieldの比較を示しています：

 |パラメータ|Seeeduino Ethernet|Ethernet Shield V1.0|Ethernet Shield V2.0|
 |---|---|---|---|
 |**動作電圧**|+6.2V~+13V|+5V|+5V|
| **制御モード**|MEGA328Pチップ搭載（Seeeduino/Arduino不要） |Seeeduino/Arduinoによる制御|Seeeduino/Arduinoによる制御|
|**TCP/IPイーサネットコントローラー**| Wiz5100| Wiz5100 |Wiz5200|
|**電源供給**| 外部電源モード| Arduino/Seeeduinoの5Vピン| Arduino/Seeeduinoの5Vピン|
|**標準シールド** |はい| いいえ |はい|
|**SDカードソケット** |FAT16またはFAT32形式のMicro SDカードをサポート、最大ストレージは2GB。 |いいえ| FAT16またはFAT32形式のMicro SDカードをサポート、最大ストレージは2GB。|
|**Groveコネクタ** |UARTおよびIICコネクタ |いいえ| UARTおよびIICコネクタ|
|**イーサネットジャック**|最小RJ45 |標準RJ45 |最小RJ45|

Gemini 335Lgは、業界で実績のあるGemini 335LのGMSL2/FAKRAバージョンとして、Gemini 330シリーズの中でも際立ったモデルです。

GMSL2シリアライザーとFAKRAコネクタの組み合わせにより、複雑な環境をナビゲートする自律移動ロボット（AMR）や柔軟なケーブル配線が必要なロボットアームに対して、安全で信頼性の高い接続を確保します。

## 仕様

<Orbbec335LgTable />

<p>[1] 理論上の最大深度範囲は65メートル；</p>
<p>[2] 工場校正時に測定されたデータ；</p>
<p>[3] Gemini 335Lgのデフォルトレーザーエネルギーレベルは、ピーク電力消費を削減し、動作不良を防ぐためにレベル4に設定されています。十分な電力が利用可能な場合、ユーザーは手動でレーザーを最大レベル6に調整して最適なパフォーマンスを実現できます；</p>
<p>[4] Gemini 335LおよびGemini 335Lgの両方とも、電源投入時の動作でIP65保護を実現するためにIP65準拠のケーブルの使用が必要です；</p>
<p>[5] 1日あたり16時間以内の動作。</p>


## デバイスのセットアップ

このチュートリアルでは、USB 3とGMSL2接続の両方を実演します。ホストデバイスの仕様：

+ reComputer Robotics J4012 with GMSL
+ JetPack 6.2 (Ubuntu 22.04)

:::note
x86_64 Windows/Linux、またはmacOS（x86_64、Apple Siliconの可能性あり）でのチュートリアルについては、[SDK Repo](https://github.com/orbbec/OrbbecSDK_v2)のREADMEを参照してください
:::

### Orbbec Viewerのダウンロードとインストール

+ Jetsonデバイスでターミナルを起動し、GitHubリリースから事前ビルドされたパッケージをダウンロードします。

``` sh
wget -L https://github.com/orbbec/OrbbecSDK_v2/releases/download/v2.4.8/OrbbecSDK_v2.4.8_arm64.deb
```

+ Install SDK

``` sh
sudo dpkg -i OrbbecSDK_v2.4.8_arm64.deb
```

<Flex justify="center">
  <Image width={600} src="https://files.seeedstudio.com/wiki/robotics/Sensor/Camera/Orbbec_Gemini_335lg/Install_Orbbec_SDK.webp" />
</Flex>

+ Orbbec Viewerを使用した可視化、ダウンロード

``` sh
wget -L https://github.com/orbbec/OrbbecSDK_v2/releases/download/v2.4.8/OrbbecViewer_v2.4.8_202507031357_a1355db_linux_aarch64.zip
```

+ コンテンツを展開する

``` sh
# In case you haven't install 'unzip'
# sudo apt-get update && sudo apt-get install unzip

unzip OrbbecViewer_v2.4.8_202507031357_a1355db_linux_aarch64.zip
```

+ Launch the Orbbec Viewer

``` sh
cd OrbbecViewer_v2.4.8_202507031357_a1355db_linux_aarch64
./OrbbecViewer
```

### GMSL2接続

カメラ上部のDIPスイッチ（まず金属カバーを取り外してください）を文字**'M'**の位置に設定します。

<Flex justify="center" style={{ 
  margin: '24px 0',
}}>
  <Image width={400} style={{
      borderRadius: 12,
      boxShadow: '0 4px 12px rgba(0,0,0,0.1)'
  }} 
  src="https://files.seeedstudio.com/wiki/robotics/Sensor/Camera/Orbbec_Gemini_335lg/GMSL2_DIP_Switch.webp" />
</Flex>

必要なもの：

+ [Mini-Fakraケーブル](https://www.seeedstudio.com/Mini-fakra-Coaxial-Cable-4-in-1-0-5m-Female-to-Female-p-6484.html)
+ [reComputer Robotics GMSL2拡張ボード](https://www.seeedstudio.com/reComputer-Robotics-GMSL-board-p-6511.html)

``` sh
sudo /opt/nvidia/jetson-io/jetson-io.py
```

このキットは**Wio Link Deluxe Kit**と同じですが、このキットにはさらに2つのディスプレイモジュールが含まれており、ユーザーに視覚的なインターフェースを追加することができます。このキットを使用して、実際のプロジェクトを実行したり、プロトタイピングを行うことができます。センサー（主に）、スイッチ機能モジュール、サーボモーター、LEDストリップ、スピーカーが含まれています。このキットの最も驚くべき部分は、Wio Link開発ボードであり、プログラミングスキルが全く必要なく、さまざまなデバイスに利用できます。モジュールに実行させたいすべてのアクションや、Wio Linkへのコードフラッシュ操作は、モバイルアプリでタップするだけで完了します。また、インターネットを介してどこからでもデバイスに指示を送ることができます。そのため、Wio Linkは周囲のものをよりスマートにするための完璧なツールとなります。さあ、何か実際に作ってみましょう！

[![](https://files.seeedstudio.com/wiki/common/Get_One_Now_Banner.png)](https://www.seeedstudio.com/depot/Wio-Link-Deluxe-Plus-Kit-p-2616.html)

:::caution
     Wio IFTTT機能はEOL（End of Life）となりました。しかし、Wio APPは利用可能です。Wio APPのAPIを使用してセンサーの状態を読み取ったり、アクチュエーターを制御することができます。
:::

特徴
--------

-   一般的に必要とされる機能モジュールのフルパッケージ
-   Groveポートインターフェース、より実用的な機能と少ない作業
-   周囲のものを簡単にスマート化し、日常生活に楽しさを追加
-   コストパフォーマンスが高い
-   重量：355 g

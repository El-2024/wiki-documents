![](https://files.seeedstudio.com/wiki/Wio_Link_Environment_Kit/img/Wio_Link_Environment_Kit_product_view_1200_s.jpg)

我们总是与周围的环境密切相关。因此，了解环境的一些细节对我们来说很重要。这个套件旨在完美地完成这项任务，至少在基础层面上如此。在这个套件中，您可以找到常见的功能模块来收集数据，例如室内的湿度或光照水平等环境数据。这个套件最令人惊叹的部分是 Wio Link 开发板，它可以用于各种用途，而无需任何编程技能。您希望模块执行的所有操作以及向 Wio Link 刷写代码的操作都可以通过移动应用程序中的点击来完成。您可以从任何地方通过互联网向您的设备发送指令。因此，Wio Link 将是一个完美的工具，让每个人都能让周围的事物变得更智能。让我们来了解一下您所处的环境条件。

[![](https://files.seeedstudio.com/wiki/common/Get_One_Now_Banner.png)](https://www.seeedstudio.com/depot/Wio-Link-Environment-Kit-p-2612.html)


:::caution
     Wio IFTTT 功能已停止服务。但 Wio APP 仍然可用。您可以使用 Wio APP 的 API 来读取传感器状态和控制执行器。
:::
特性
--------

-   包含常用的环境监测功能模块
-   Grove 接口，更实用的功能，更少的工作量
-   性价比高
-   重量：154 g

# 将 Wio Terminal 连接到 Microsoft Azure IoT

<div align="center"><img src="https://files.seeedstudio.com/wiki/Wio-Terminal-Azure-IOT/Wio-terminal-azureiot.jpeg" /></div>


>这是一份由 [**@Benjamin Cabé**](https://twitter.com/kartben) 编写的文档。感谢 Benjamin 的出色工作！您也可以在[这里](https://github.com/kartben)找到 Benjamin 的作品。

这个示例应用程序向您展示如何将您的 [Wio Terminal](https://www.seeedstudio.com/Wio-Terminal-p-4509.html) 连接到 [Azure IoT Hub](https://azure.microsoft.com/services/iot-hub)。它基于 [Azure SDK for Embedded C](https://github.com/Azure/azure-sdk-for-c) 构建，这是一个占用空间小、易于移植的库，用于与 Azure 服务通信。

要将 Wio Terminal 与 Microsoft Azure IoT 连接，请参阅 [**Benjamin 的指南**](https://github.com/kartben/wioterminal-azureiothub-sample)。

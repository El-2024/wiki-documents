此 Grove 模块提供了 [逻辑非门](https://en.wikipedia.org/wiki/Inverter_%28logic_gate%29) 的功能。它基于 74HC1G04GW 芯片。74HC1G04GW 是一种高速 Si 门 CMOS 器件，提供反向缓冲功能。非门模块左侧有一个输入，右侧有一个输出。当输入信号为高电平时，输出信号为低电平；当输入信号为低电平时，输出信号为高电平。

[![](https://files.seeedstudio.com/wiki/Seeed-WiKi/docs/images/300px-Get_One_Now_Banner-ragular.png)](https://www.seeedstudio.com/Mixer-Pack-V2-(Electronic-blocks%2Cwithout-Arduino%2Cplug-and-play-system)-p-1867.html)

## 特性

*   简单的非门功能，具有一个输入和一个输出

*   使在电路中集成逻辑非门变得轻而易举

*   使用标准 4 针 Grove 电缆连接到其他 Grove 模块

## 使用方法

*   请参阅 Grove Mixer Pack V2 Wiki 页面上的发光 LED 串灯示例，了解如何构建使用非门逻辑的电路。

*   另请参阅 Grove - 非门模块的典型使用方法

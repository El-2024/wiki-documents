![](https://files.seeedstudio.com/wiki/RF_Explorer_Software/img/RF%20Explorer.jpg)

RF Explorer 是一款卓越的诊断工具，用于监控和排查无线系统及通信问题。它是一系列由 Ariel 设计的低成本、便携式 RF 频谱分析仪，提供多种频率范围选择。

RF Explorer 系列分析仪和信号发生器重新定义了手持便携式 RF 频谱分析仪。该设备体积小巧、重量轻，但性能和功能可媲美更大、更昂贵的设备。

过去，RF 频谱分析仪的使用仅限于能够负担高昂价格的公司或组织，而 RF Explorer 的低成本现在让每位工程师、技术人员和精明的爱好者都有机会拥有一台 RF 频谱分析仪。

以下是一些您可能感兴趣的软件应用程序——它们可在多个平台上运行。这使您能够尝试软件提供的各种功能，并确定哪款最适合您的需求。有关更多详细信息以及适用于所有平台的 RF Explorer 兼容软件的完整列表，请访问 [这里](http://www.rf-explorer.com/software)。

## RF Explorer 的 Windows 软件选项

### [RF Explorer for Windows](http://www.rf-explorer.com/windows)  

![](https://files.seeedstudio.com/wiki/RF_Explorer_Software/img/Windows.jpg)

来自 [RF Explorer 开发者](http://www.rf-explorer.com) 的 RF Explorer for Windows 应用程序提供传统的频谱轨迹视图以及更具图形化的瀑布视图。它还包括运行信号发生器和作为标量网络分析仪所需的所有软件。

Nuts About Nets 开发了两种不同版本的软件——标准版和专业版。**[标准版](http://rfexplorer.com/touchstone)** 是免费的，而 **[专业版](http://rfexplorer.com/touchstone)** 仅需 49 美元。

来自 [Nuts About Nets](http://rfexplorer.com/) 的 Touchstone 软件将 RF Explorer 频谱分析仪收集的数据转化为高度图形化的图表和显示，使用户能够更容易地可视化 RF 环境、监控 RF 信号、排查 RF 问题以及检测 RF 干扰源。

## MacOS RF Explorer 软件选项

### [iRF Explorer for MacOS X - 免费](https://itunes.apple.com/app/irfexplorer/id474348349?mt=12)
![](https://files.seeedstudio.com/wiki/RF_Explorer_Software/img/Mac.jpeg)
可在 Mac Store 中获取，它为 Mac 提供基本且易于使用的标准频谱分析仪功能。支持 Mac OS X 10.7 及更高版本，请确保按照安装说明安装 Silicon Labs 驱动程序。

### [Vantage - $99 USD](http://www.rfvenue.com/vantage)
![](https://files.seeedstudio.com/wiki/RF_Explorer_Software/img/rfvenue.jpg)
为 Mac 提供强大且易于使用的标准频谱分析仪功能，支持所有 RF Explorer 型号。包括导出文件功能，可用于主要的第三方协调软件工具。未来还将推出高级功能。

## Android RF Explorer 软件选项

### [RFTrack for Android - 免费](http://wireless.ictp.it/tvws/rftrack/)

适用于支持 OTG 的 Android 手机和平板电脑，它结合了 Android 的 GPS 位置和 RF Explorer 的 RF 活动，用于创建全球 TVWS 数据库。

### [Touchstone Mobile - $49 USD](http://rfexplorer.com/touchstone-mobile)

![](https://files.seeedstudio.com/wiki/RF_Explorer_Software/img/touchstone-mobile.jpg)

适用于支持 OTG 的 Android 手机和平板电脑，它将 RF Explorer 频谱分析仪收集的数据转化为高度图形化的图表和显示，帮助用户更直观地了解 RF 环境、监控 RF 信号、排查 RF 问题以及检测 RF 干扰源。

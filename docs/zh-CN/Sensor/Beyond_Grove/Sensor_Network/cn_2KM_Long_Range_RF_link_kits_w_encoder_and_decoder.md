![](https://files.seeedstudio.com/wiki/2KM_Long_Range_RF_link_kits_w_encoder_and_decoder/img/2kmrf.jpg)

这是一个超长距离的 433MHz RF 链路套件。它采用 VCO 和 PLL 技术，具有稳定的频率和超强的抗干扰能力。您可以直接将其用于您的项目，例如无线数据传输、远程控制等。

[![](https://files.seeedstudio.com/wiki/Seeed-WiKi/docs/images/300px-Get_One_Now_Banner-ragular.png)](https://www.seeedstudio.com/2km-long-range-rf-link-kits-w-encoder-and-decoder-p-321.html?cPath=139_140)

##   规格
---
| 参数                   | 值                                       |
|------------------------|------------------------------------------|
| 工作电压              | 接收器(3-5V)，发射器(3-9V)               |
| 工作电流              | ≤2.5mA(5.0VDC)                           |
| 工作原理              | 超外差（VCO，PLL）                       |
| 调制方式              | OOK/ASK                                  |
| 工作频段              | 433.92MHz（可定制服务）                  |
| 带宽                  | 1.5MHz                                   |
| 灵敏度                | -105dBm (50Ω)                            |
| 速率                  | ＜5Kbps                                  |
| 解码形式              | PT2272                                   |
| 天线长度              | 18cm                                     |
| 传输范围              | 2 km                                     |

##   应用场景
---
*   无线开关
*   远程传输
*   无线控制

## 入门指南
---

- 请按照 [说明书](https://files.seeedstudio.com/wiki/2KM_Long_Range_RF_link_kits_w_encoder_and_decoder/res/2KM_RF.rar) 设置通信系统。
- 连接电池、按钮和发射板。
![](https://files.seeedstudio.com/wiki/2KM_Long_Range_RF_link_kits_w_encoder_and_decoder/img/2KM_TX.JPG)

!!!注意
    请使用 3-5V 电源！9V 电源会损坏接收板。
- 连接电池、LED 和接收板。
![](https://files.seeedstudio.com/wiki/2KM_Long_Range_RF_link_kits_w_encoder_and_decoder/img/2KM_RX.JPG)

!!!注意
    请勿将接收模块和发射模块放得太近：这会导致它们无法正常工作。确保接收模块和发射模块至少相距 1 米。
- 当我们按下发射端的按钮时，接收端的相关 LED 会被点亮。

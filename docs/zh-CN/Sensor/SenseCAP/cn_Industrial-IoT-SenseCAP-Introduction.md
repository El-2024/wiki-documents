## 野外物联网 (Wild IoT)

野外物联网是指在自然环境中部署传感器网络，以实时监测和收集各种环境数据。这些数据包括温度、湿度、气压、光照强度、土壤湿度等，有助于科学研究、生态保护和资源管理。

### 为什么需要工业传感器？

1. **耐用性和可靠性**：户外环境复杂多变。工业传感器具有耐高温、高湿和抗腐蚀的特点，能够在恶劣条件下长期稳定运行。
2. **高精度数据采集**：科学研究和环境监测需要高精度数据。工业传感器的高精度和稳定性满足了这些需求。
3. **低功耗和长电池寿命**：在野外供电可能是一个挑战。工业传感器通常设计为低功耗设备，能够长时间运行，减少维护和更换电池的频率。

### 它们如何工作？

1. **数据采集**：工业传感器安装在户外环境中，实时采集温度、湿度、气压等数据。
2. **数据传输**：采集到的数据通过低功耗广域网 (LPWAN) 技术（如 LoRaWAN）传输到网关。
3. **数据处理和存储**：网关将数据发送到云平台进行处理、存储和分析。
4. **数据可视化和应用**：用户可以通过云平台上的应用程序或仪表板实时查看数据，辅助分析和决策。

### SenseCAP API

通过使用 SenseCAP 应用程序编程接口，您可以管理自己的 SenseCAP 设备和数据，以满足您的特定需求。您可以灵活应用以下三种类型的 API 方法：

* 使用 HTTP API，您可以管理 LoRa 和 NB-IoT 设备，获取原始数据或历史数据。
* 使用 MQTT API，您可以通过 MQTT 协议订阅传感器的实时测量数据。
* 使用 WebSocket API，您可以通过 WebSocket 协议获取传感器的实时测量数据。

要了解更多关于 SenseCAP API 的指南和参考，请访问 [SenseCAP 文档中心 – API](https://sensecap-docs.seeed.cc/introduction.html)。

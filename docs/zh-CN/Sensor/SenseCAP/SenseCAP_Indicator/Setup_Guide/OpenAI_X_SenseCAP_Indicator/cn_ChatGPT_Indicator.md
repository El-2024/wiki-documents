  # **SenseCAP Indicator X ChatGPT**

ChatGPT 是由 OpenAI 开发的对话式 AI 模型。它专门为聊天机器人应用而设计，并在大量对话数据上进行了训练，使其能够理解并生成对用户输入的自然语言响应。ChatGPT 已被用于各种应用，包括客户服务聊天机器人、虚拟助手和社交媒体聊天机器人。


SenseCAP Indicator 通过 OpenAI 的 API 将 ChatGPT 集成到原生固件中，它可以帮助您快速体验与 ChatGPT 的对话。


## 步骤 1. Wi-Fi 连接

选择 Wi-Fi，输入密码并连接。


<div align="center"><img width={480} src="https://files.seeedstudio.com/wiki/SenseCAP/SenseCAP_Indicator/SenseCAP_Indicator_9.png"/></div>


<div align="center"><img width={400} src="https://files.seeedstudio.com/wiki/SenseCAP/SenseCAP_Indicator/SenseCAP_Indicator_10.png"/></div>


## 步骤 2. 设置 API 密钥

点击"Set API Key"并扫描二维码，然后根据[教程](/cn/Sensor/SenseCAP/SenseCAP_Indicator/Set_An_API_Key)进行设置。

<div align="center"><img width={480} src="https://files.seeedstudio.com/wiki/SenseCAP/SenseCAP_Indicator/apikey.png"/></div>


## 步骤 3. 开始与 ChatGPT 通信

当您设置好 Wi-Fi 和 APIKey 后，一切就准备就绪了。

**向 ChatGPT 询问您想知道的任何事情！**

除了演示的功能外，SensecAP 还提供完全开源的代码和详细的教程，点击[这里](/cn/SenseCAP_Indicator_ChatGPT)了解更多详情。


# **常见问题**

**设备已连接到 WIFI 但仍提示"请检查 wifi 连接"**

- 请尝试连接到另一个 Wi-Fi，或检查您的路由器设置。


**请先设置 Open AI API 密钥**

- API 密钥未设置，或设置失败，请重新设置。


**请求失败**

- 检查 API 密钥是否正确。

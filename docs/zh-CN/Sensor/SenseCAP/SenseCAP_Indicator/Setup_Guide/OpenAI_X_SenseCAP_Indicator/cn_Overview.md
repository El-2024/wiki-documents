SenseCAP 指示器是一款强大的工具，通过 API 集成利用了 OpenAI 的 ChatGPT 和 DALL·E 模型的能力。由人工智能领域知名的 OpenAI 开发，SenseCAP 指示器让用户能够在各种场景中体验 AI 的迷人潜力。

SenseCAP 指示器集成了 [ChatGPT](https://platform.openai.com/docs/guides/chat) 和 [DALL·E](https://platform.openai.com/docs/guides/images/introduction) AI 服务。它是一款多功能的智能助手，支持实时对话、语言学习与翻译以及有趣的基于文本的游戏等无缝交互。

开发者还可以利用 SenseCAP 指示器将 AI 集成到他们的应用中，从 **智能家居控制**、**IFTTT 自动化到传感器数据分析**等应用场景。

## 开始使用

要开始使用 SenseCAP 指示器，请按照以下步骤操作：

1. 按下界面上的“ChatGPT”按钮。
2. 在输入框中输入您的文本提示，例如“骆驼是什么？”。
3. 隐藏键盘（按 `√`），然后点击“生成”按钮。
4. 等待响应，结果将显示在屏幕上。

<div align="center"><img width={680} src="https://files.seeedstudio.com/wiki/SenseCAP/SenseCAP_Indicator/GPT_REQ_WHAT_IS_CAM.JPG"/></div>

您还可以测试 ChatGPT 的数学能力，并探索其回答各种问题的能力。

<div align="center"><img width={680} src="https://files.seeedstudio.com/wiki/SenseCAP/SenseCAP_Indicator/chatgpt.jpg"/></div>

同样，DALL·E 集成提供了图像生成功能。通过提供文本提示，您可以根据描述请求生成的图像。

<div align="center"><img width={680} src="https://files.seeedstudio.com/wiki/SenseCAP/SenseCAP_Indicator/DALL_ONECAT_WHOLE.jpg"/></div>

如果您对生成的图像不满意，可以点击“重新生成”按钮请求另一个图像。

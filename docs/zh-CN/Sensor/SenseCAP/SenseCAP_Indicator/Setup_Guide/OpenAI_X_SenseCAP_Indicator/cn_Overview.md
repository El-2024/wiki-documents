SenseCAP Indicator 是一个强大的工具，通过 API 集成利用 OpenAI 的 ChatGPT 和 DALL·E 模型的功能。由人工智能领域的知名公司 OpenAI 开发，SenseCAP Indicator 让用户能够在各种场景中体验 AI 的迷人潜力。

SenseCAP Indicator 集成了 [ChatGPT](https://platform.openai.com/docs/guides/chat) 和 [DALL·E](https://platform.openai.com/docs/guides/images/introduction) AI 服务。它作为一个多功能智能助手，支持无缝交互，如实时对话、语言学习和翻译，以及引人入胜的文本游戏。

开发者还可以利用 SenseCAP Indicator 将 AI 集成到他们的应用程序中，涵盖**智能家居控制**和 **IFTTT 自动化到传感器数据分析**等各种场景。

## 功能特性

OpenAI 的 ChatGPT 和 DALL·E 模型与 SenseCAP Indicator 的集成提供以下关键功能：

### 语言处理

**ChatGPT** 凭借其先进的自然语言处理能力，实现智能和类人的对话流程。用户可以与 SenseCAP Indicator 进行有意义的对话，利用其理解和生成文本的能力。

**DALL·E** 是一个图像生成模型，使 SenseCAP Indicator 能够根据文本描述创建视觉内容。用户可以通过提供文本提示并根据其输入接收生成的图像来体验 AI 的创造可能性。

## 开始使用

要开始使用 SenseCAP Indicator，请按照以下步骤操作：

1. 在界面上按下"ChatGPT"按钮。
2. 在输入字段中输入您的文本提示，例如"什么是骆驼？"。
3. 隐藏键盘（按 `√`）并点击"Generate"按钮。
4. 等待响应，它将显示在屏幕上。

<div align="center"><img width={680} src="https://files.seeedstudio.com/wiki/SenseCAP/SenseCAP_Indicator/GPT_REQ_WHAT_IS_CAM.JPG"/></div>

您还可以测试 ChatGPT 的数学能力并探索其回答各种问题的能力。

<div align="center"><img width={680} src="https://files.seeedstudio.com/wiki/SenseCAP/SenseCAP_Indicator/chatgpt.jpg"/></div>

同样，DALL·E 集成提供图像生成功能。通过提供文本提示，您可以根据您的描述请求生成图像。

<div align="center"><img width={680} src="https://files.seeedstudio.com/wiki/SenseCAP/SenseCAP_Indicator/DALL_ONECAT_WHOLE.jpg"/></div>

如果您对生成的图像不满意，可以点击"Regenerate"按钮请求另一张图像。

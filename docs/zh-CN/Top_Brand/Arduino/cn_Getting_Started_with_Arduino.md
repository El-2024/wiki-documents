## Arduino Cores

- **目的**：Arduino cores 允许 Arduino IDE 与不同类型的微控制器通信。它们将标准的 Arduino 命令适配到底层硬件的具体特性。
- **示例**：不同的 cores 适用于不同的微控制器，如 RP2040、ESP32、ESP8266，以及更传统的 AVR 和 SAMD 等。一些 cores 由社区维护，确保为广泛的设备提供支持和更新。

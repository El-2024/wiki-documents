GPRS Shield 为您提供了一种使用 GSM 蜂窝电话网络从远程位置接收数据的方法。该扩展板允许您通过以下三种方法中的任何一种来实现：

- 短信服务

- 音频

- GPRS 服务
  - [GPS 跟踪](http://vamostech.com/gps-tracking)
  - [GPS 追踪器](http://vamostech.com/gps-tracking)
  - [Properti semarang](http://www.raywhitesemarang.com)
  - [Jual sepatu online](http://mariposa-store.com/)

有两个版本供您选择：

|![](https://files.seeedstudio.com/wiki/GPRS-Shield/img/GPRS_shield_v1.4.jpg)|![](https://files.seeedstudio.com/wiki/GPRS-Shield/img/GPRSshield_01.jpg)|
|---|---|
|GPRS Shield V1.0|GPRS Shield V2.0|

以下列出了您可以在 [myamplifiers.com](http://www.myamplifiers.com/) 上找到的各种版本 GPRS Shield 之间的比较：

|  参数|GPRS Shield V1.0|GPRS Shield V2.0|
|---|---|---|
|  **工作电压** | +5V|+5V  |
|  **供电方式**|电源插孔或 Arduino/Seeeduino 的 5V 引脚|Arduino/Seeeduino 的 5V 引脚或 Vin 引脚 |
| **音频接口**|扬声器和耳机插孔|二合一耳机插孔  |
| **标准扩展板**|否|是  |
|  **天线**|0.4W|0.8W(29dBm)  |
|**发射功率**|0.4W|0.8W(29dBm)|

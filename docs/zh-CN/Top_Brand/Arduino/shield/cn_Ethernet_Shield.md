Arduino 以太网扩展板允许 Arduino 开发板连接到互联网。它基于 Wiznet W5100/W5200 以太网芯片，提供支持 TCP 和 UDP 的网络（IP）协议栈。使用以太网库编写草图，通过 RJ45 以太网接口连接到互联网。

|![](https://files.seeedstudio.com/wiki/Ethernet_Shield/img/Seeeduino_ethernet-2.jpg)|![](https://files.seeedstudio.com/wiki/Ethernet_Shield/img/Ethernet_01.jpg)|![](https://files.seeedstudio.com/wiki/Ethernet_Shield/img/W5200_Ethernet_Shield.jpg)|
|---|---|---|
|[Seeeduino 以太网](https://seeeddoc.github.io/Seeeduino_Ethernet/) |[以太网扩展板 V1.0](https://seeeddoc.github.io/Ethernet_Shield_V1.0/) |[以太网扩展板 V2.0](https://seeeddoc.github.io/Ethernet_Shield_V2.0/)|

以下列出了各版本以太网扩展板的比较：

 |参数|Seeeduino 以太网|以太网扩展板 V1.0|以太网扩展板 V2.0|
 |---|---|---|---|
 |**工作电压**|+6.2V~+13V|+5V|+5V|
| **控制模式**|配备 MEGA328P 芯片（无需 seeeduino/Arduino） |由 seeeduino/Arduino 控制|由 seeeduino/Arduino 控制|
|**TCP/IP 以太网控制器**| Wiz5100| Wiz5100 |Wiz5200|
|**供电方式**| 外部供电模式| Arduino/Seeeduino 的 5V 引脚| Arduino/Seeeduino 的 5V 引脚|
|**标准扩展板** |是| 否 |是|
|**SD 卡插槽** |支持 FAT16 或 FAT32 格式的 Micro SD 卡，最大存储容量为 2GB。 |否| 支持 FAT16 或 FAT32 格式的 Micro SD 卡，最大存储容量为 2GB。|
|**Grove 连接器** |UART 和 IIC 连接器 |否| UART 和 IIC 连接器|
|**以太网接口 **|迷你 RJ45 |标准 RJ45 |迷你 RJ45|

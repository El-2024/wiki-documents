Sidekick 高级套件包含一套基础元件以及大量传感器和其他模块。

我们为传感器章节准备了一个示例代码集。你可以在[这里](https://github.com/Seeed-Studio/sketch_sidekick)下载。

当你下载完代码后，将其解压到 F: 文件夹，或任何你喜欢的位置。

然后打开 Arduino IDE，点击 File -&gt; Preferences：

![](https://files.seeedstudio.com/wiki/Sketchbook_of_Sidekick_Advanced_Kit/img/Sidekick_40_1.png)

将 Sketchbook location 设置为你下载的代码路径，点击 OK 完成设置。重新打开 Arduino IDE。

点击 File -&gt; Sketchbook，然后你就可以找到一些项目了，这些就是本章节的演示代码。

![](https://files.seeedstudio.com/wiki/Sketchbook_of_Sidekick_Advanced_Kit/img/Sidekick_40_2.png)

**上传代码的具体步骤：**

1. 在打开示例代码后，通过以下路径选择您正在使用的Arduino开发板类型：工具 --&gt; 开发板 --&gt; 例如，Arduino UNO。
![](https://files.seeedstudio.com/wiki/Upload_Code/img/Open_code.jpg)

如果出现下图所示的情况，可能是您选择的开发板类型有误，请重新选择您正在使用的Arduino开发板。

![](https://files.seeedstudio.com/wiki/Upload_Code/img/Error_score.jpg)

2. 通过以下路径选择您正在使用的正确串口：工具 --&gt; 串口 --&gt; 例如，COM3。
3. 上传示例代码，如下图所示：

![](https://files.seeedstudio.com/wiki/Upload_Code/img/Upload_state.jpg)

当出现"上传完成"时，恭喜您上传成功。

有些人认为Arduino IDE的编辑界面不够友好，缺乏标准缩进、语法高亮、自动补全等功能。简而言之，如果你使用过其他优秀的编辑器，如emacs、vim，甚至notepad++，你一定会对Arduino的编辑界面感到不满。

但是，你可以使用外部编辑器，打开Arduino IDE，点击文件 -&gt; 首选项，选择使用外部编辑器：

![](https://files.seeedstudio.com/wiki/Use_External_Editor/img/外部编辑器1.png)

然后你会发现无法在Arduino IDE中输入，现在只能使用外部编辑器了。

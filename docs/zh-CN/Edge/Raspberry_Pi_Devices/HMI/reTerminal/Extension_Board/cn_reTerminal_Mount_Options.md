# reTerminal 安装选项

## 介绍

reTerminal 作为一个优秀的人机界面（HMI），已被广泛应用于许多不同的系统中。作为系统的接口，安装 reTerminal 而不是仅仅手持使用会是一个好主意。
在本节中，我们将讨论 reTerminal 安装解决方案的可能性。本节内容将涵盖[硬件螺丝规格](https://wiki.seeedstudio.com/cn/reTerminal_Mount_Options/#Hardware_Specification_of_the_Screw)和[社区设计](https://wiki.seeedstudio.com/cn/reTerminal_Mount_Options/#Community_Design)。

## 硬件螺丝规格

设备背面和侧面的所有嵌入式螺母都适配 M4 螺丝，设备中间的 2 个螺母适配 **M4 18mm** 螺丝，其他的适配 **M4 6mm** 螺丝。
底部的安装孔是直径为 ¼ 英寸的**标准相机安装孔**，因此您可以将 reTerminal 连接到标准三脚架上。

## 社区设计

### ReTerminal 旋转模块 "Rota Terminal"

这个[旋转模块](https://protopedia.net/prototype/2903)允许用户旋转和固定 reTerminal，配合新固件，当屏幕旋转到垂直位置时，屏幕会自动调整为垂直方向。
此外，通过进一步开发，这个模块还有更多可能性。
*参考：[https://protopedia.net/prototype/2903](https://protopedia.net/prototype/2903)*
<p style={{textAlign: 'center'}}><img src="https://files.seeedstudio.com/wiki/reTerminal_Mount_Options/5.png" alt="pir" width="1000" height="auto"/></p>

### reTerminal 天花板模块

[reTerminal 天花板模块](https://protopedia.net/prototype/2943)是一个将 reTerminal 转换为物联网控制设备的项目，该模块的功能包括控制插座的电源供应和通过摄像头进行监控。
*参考：[https://protopedia.net/prototype/2943](https://protopedia.net/prototype/2943)*

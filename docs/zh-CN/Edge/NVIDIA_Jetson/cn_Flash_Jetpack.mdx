### 3. 安装刷写依赖
在主机 PC 上安装刷写依赖：
```bash
sudo apt-get update && \
sudo apt-get install -y abootimg \
                        binfmt-support \
                        binutils \
                        cpp \
                        device-tree-compiler \
                        dosfstools \
                        lbzip2 \
                        libxml2-utils \
                        nfs-kernel-server \
                        python3-yaml \
                        qemu-user-static \
                        sshpass \
                        udev \
                        uuid-runtime \
                        whois \
                        openssl \
                        cpio \
                        rsync \
                        zstd
```

此外，对于 Ubuntu 18.04 及以下版本，安装 `lz4c`

```bash
sudo apt-get install -y lz4
```

对于 Ubuntu 20.04 及以上版本：

```bash
sudo apt-get install -y lz4
```

### 4. 解压和刷写

### 4. 安装刷写依赖项
在主机 PC 上安装刷写依赖项：
```bash
sudo apt-get update && \
sudo apt-get install -y abootimg \
                        binfmt-support \
                        binutils \
                        cpp \
                        device-tree-compiler \
                        dosfstools \
                        lbzip2 \
                        libxml2-utils \
                        nfs-kernel-server \
                        python3-yaml \
                        qemu-user-static \
                        sshpass \
                        udev \
                        uuid-runtime \
                        whois \
                        openssl \
                        cpio \
                        rsync \
                        zstd
```

另外，对于 Ubuntu 18.04 及以下版本，请安装 `lz4c`。

```bash
sudo apt-get install -y liblz4-tool
```

对于 Ubuntu 20.04 / 22.04 及更高版本：

```bash
sudo apt-get install -y lz4
```

此外壳由军用级铝材和耐用的亚克力制成，提供了极致的物理和防腐保护！骨架盒为任何标准 Arduino 留出了空间，并且外壳可以在三个维度上堆叠，从而实现无限的扩展可能性。骨架盒易于使用，为您的 Arduino 提供了最好的保护！这是一个简单的外壳，专为 BeagleBone® 设计。

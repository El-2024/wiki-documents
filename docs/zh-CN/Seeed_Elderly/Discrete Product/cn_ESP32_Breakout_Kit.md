这是专为 ESP32 设计的扩展套件，包含一个扩展板和针脚排针。尺寸为 53mmx38mmx1.2mm。扩展板配备 2.54mm 间距的母排针，方便您轻松连接 ESP32 的针脚。我们还在此套件中提供了 2 个 Grove 接口，以增强 ESP32 的扩展性。

[![enter image description here](https://files.seeedstudio.com/wiki/Seeed-WiKi/docs/images/get_one_now.png)](https://www.seeedstudio.com/ESP32-Breakout-Kit-p-2783.html)

## 部件清单

:::note
    套件中的部件未焊接。
:::
![enter image description here](https://files.seeedstudio.com/wiki/ESP32_Breakout_Kit/img/part_list.jpg)


|部件名称|数量|
|---------|---|
|4Pin Grove 接口|2|
|4Pin 2.54 公排针|2|
|10Pin 2.54 公排针|4|
|10Pin 2.54 母排针|4|
|PCB|1|

## 入门指南

- 首先，您需要购买 [ESP-32S Wifi Bluetooth Combo Module](https://www.seeedstudio.com/ESP-32S-Wifi-Bluetooth-Combo-Module-p-2706.html)。
![enter image description here](https://files.seeedstudio.com/wiki/ESP32_Breakout_Kit/img/ESP32.jpg)
- 其次，将 ESP32 模块焊接到扩展套件上。
- 组装完成的套件将提供超高性能和超低功耗的 Wi-Fi 和蓝牙组合无线解决方案。

<!-- ### Q1: 默认用户 "recomputer" 的密码是什么？ -->

系统已预设了用户名和密码。请在登录时将默认用户名设置为 "**recomputer**"，默认密码设置为 "**12345678**"。如果您设置了不同的凭据，可能会遇到问题。目前该问题正在解决中。

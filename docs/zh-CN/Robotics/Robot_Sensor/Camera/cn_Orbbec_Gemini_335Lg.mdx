Gemini 335Lg 是 Gemini 330 系列中的杰出型号，作为业界验证的 Gemini 335L 的 GMSL2/FAKRA 版本。

GMSL2 串行器结合 FAKRA 连接器确保了在复杂环境中导航的自主移动机器人（AMR）和需要灵活布线的机械臂的安全可靠连接。

## 规格参数

<Orbbec335LgTable lang="zh" />

<p>[1] 理论最大深度范围可达 65 米；</p>
<p>[2] 工厂校准期间测量的数据；</p>
<p>[3] Gemini 335Lg 默认激光能量级别设置为 4 级，以降低峰值功耗并防止操作故障。当有足够功率可用时，用户可以手动将激光调整到最大 6 级以获得最佳性能；</p>
<p>[4] Gemini 335L 和 335Lg 在通电操作期间都需要使用符合 IP65 标准的电缆才能实现 IP65 防护；</p>
<p>[5] 每天工作时间不超过 16 小时。</p>


## 设置设备

在本教程中，我们演示 USB 3 和 GMSL2 连接。主机设备规格：

+ 带 GMSL 的 reComputer Robotics J4012
+ JetPack 6.2（Ubuntu 22.04）

:::note
有关 x86_64 Windows/Linux 或 MacOS（x86_64，可能包括 Apple Silicon）的教程，请参考 [SDK 仓库](https://github.com/orbbec/OrbbecSDK_v2) 的 README
:::

### 下载并安装 Orbbec Viewer

+ 在您的 Jetson 设备上启动终端，从 GitHub 发布页面下载预构建包。

``` sh
wget -L https://github.com/orbbec/OrbbecSDK_v2/releases/download/v2.4.8/OrbbecSDK_v2.4.8_arm64.deb
```

+ 安装 SDK

``` sh
sudo dpkg -i OrbbecSDK_v2.4.8_arm64.deb
```

<Flex justify="center">
  <Image width={600} src="https://files.seeedstudio.com/wiki/robotics/Sensor/Camera/Orbbec_Gemini_335lg/Install_Orbbec_SDK.webp" />
</Flex>

+ 使用 Orbbec Viewer 进行可视化，下载

``` sh
wget -L https://github.com/orbbec/OrbbecSDK_v2/releases/download/v2.4.8/OrbbecViewer_v2.4.8_202507031357_a1355db_linux_aarch64.zip
```

+ 解压内容

``` sh
# In case you haven't install 'unzip'
# sudo apt-get update && sudo apt-get install unzip

unzip OrbbecViewer_v2.4.8_202507031357_a1355db_linux_aarch64.zip
```

+ 启动 Orbbec Viewer

``` sh
cd OrbbecViewer_v2.4.8_202507031357_a1355db_linux_aarch64
./OrbbecViewer
```

+ [Mini-Fakra 电缆](https://www.seeedstudio.com/Mini-fakra-Coaxial-Cable-4-in-1-0-5m-Female-to-Female-p-6484.html)

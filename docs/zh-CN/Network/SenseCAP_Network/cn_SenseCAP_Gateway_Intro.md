## 介绍

**SenseCAP 网关**是一款工业级、高性能的网关，专为长距离无线通信而设计。由矽递科技开发的 SenseCAP 网关作为物联网（IoT）生态系统中的关键组件，为广泛的应用提供稳健可靠的连接。

<p style={{textAlign: 'center'}}><img src="https://files.seeedstudio.com/wiki/SenseCAP/introduction/helium-sensecap6.png" alt="pir" width={1000} height="auto" /></p>

SenseCAP 网关的主要特性包括：

- 高可靠性：采用 IP66 级防护外壳，能够承受恶劣的环境条件。
- 长距离：利用 LoRa 技术能够提供广泛的覆盖范围。
- 高性能：配备强大的处理能力来管理众多连接设备。
- 易于部署：简单的设置和维护，使各种用户和行业都能轻松使用。
- 可扩展性：支持大规模物联网部署，具有高数据吞吐量和众多设备连接能力。

SenseCAP 网关的应用范围从智慧农业和环境监测到工业自动化和智慧城市。

## LoRa 通信

LoRa（长距离）是一种低功耗广域网（LPWAN）协议，专为区域、国家或全球网络中的无线电池供电设备而设计。LoRa 非常适合需要长距离、低功耗和安全数据传输的应用。

1. 调制技术：LoRa 使用啁啾扩频（CSS）调制，将信号扩展到宽频率范围内，提供对干扰和信号衰减的强大抗性。
2. 长距离：LoRa 可以在农村地区传输数据达 15 公里，在城市地区达 5 公里，使其适用于广泛的应用。
3. 低功耗：使用 LoRa 的设备由于低功耗要求，可以用小电池运行数年。
4. 低数据速率：虽然 LoRa 支持长距离通信，但它针对低数据速率进行了优化，通常范围从 0.3 kbps 到 50 kbps。
5. 安全通信：LoRa 提供强大的安全功能，包括端到端加密，确保数据完整性和机密性。
6. LoRa 通常使用星形拓扑结构，其中网关在终端设备和中央网络服务器之间中继消息。网关充当透明桥梁，将射频数据包转换为 IP 数据包，反之亦然。

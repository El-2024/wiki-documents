**什么是 SenseCAP M2 数据专用**

SenseCAP M2 数据专用是一款连接到 Helium LongFi 网络的 LoRaWAN 网关，能够为长距离/LoRaWAN 设备提供数英里的无线网络覆盖和数据传输能力。

作为一款 LoRaWAN 网关，它能够为长距离/LoRaWAN 设备提供数英里的无线网络覆盖和数据传输能力。

作为支持 Helium LongFi 网络的网关，它可以帮助您以低成本构建本地 LoRaWAN 网络并开发各种 LoRaWAN 应用，同时享受成为“人民网络”一部分的好处。

SenseCAP M2 目前提供 US915 和 EU868 版本，并通过了 FCC 和 CE 认证。未来将为不同国家和地区提供更多频段版本及相应的认证。

**功能特点**
============

![](https://www.sensecapmx.com/wp-content/uploads/2022/07/sensecap-m2-DO-features.png)

**· 广覆盖范围和强信号**

配备 3dBi 高性能天线，SenseCAP M2 可提供数英里的 LoRaWAN 覆盖和强信号。您还可以根据不同环境升级天线以适应您的使用场景。

**· 优异且稳定的性能**

SenseCAP M2 采用成熟的硬件解决方案 MT7628 和 Semtech SX1302/SX1303 基带长距离芯片。它内置 BLE，并支持 Wi-Fi 和以太网连接。对于需要通过以太网供电而不是额外电源线的用户，M2 还增加了 PoE 功能。

**· 专业管理工具和云服务**

用户可以通过 SenseCAP Hotspot App 轻松完成热点设置。SenseCAP Hotspot Dashboard 和 SenseCAP Local Console 也为用户提供了高效、便捷的热点监控和管理方式。

**· 高效及时的在线支持**

除了这些专业管理工具和云服务，SenseCAP 的技术支持也备受客户好评。无论何时遇到技术问题，高效的在线技术支持服务始终在这里为您提供及时帮助。

**· 安全可靠**  
对于处理大量数据传输的网关来说，安全性永远是重中之重。SenseCAP M2 数据专用始终确保高安全认证和可靠的“人民网络”连接。

**· 简单步骤轻松设置**

通过 SenseCAP Hotspot App 准备就绪，简单几步即可完成设置。

**· 丰富的配件**

除了 SenseCAP 设备本身的强大功能外，SenseCAP 还为用户提供了丰富的配件选择，以获得热点的最佳性能。例如，高性能物联网天线、天线电缆、安装垫和避雷器等畅销配件。

=============

![](https://www.sensecapmx.com/wp-content/uploads/2022/07/插图-01.jpg)

**注意：**

*   MicroSD 卡槽为未来使用预留。作为 LoRaWAN 网关，SenseCAP M2 不需要 Micro SD 卡。
*   Nano SIM 卡槽仅适用于 4G 版本热点。设备中不包含 SIM 卡。

**什么是 SenseCAP M2 Data Only**

SenseCAP M2 Data Only 是一款连接到 Helium LongFi 网络的 LoRaWAN 网关，能够为长距离/LoRaWAN 设备提供数英里的无线网络覆盖和数据传输能力。

作为 LoRaWAN 网关，它能够为长距离/LoRaWAN 设备提供数英里的无线网络覆盖和数据传输能力。

作为支持 Helium LongFi 网络的网关，它可以帮助您以低成本构建本地 LoRaWAN 网络并开发各种 LoRaWAN 应用，同时享受作为人民网络一部分的好处。

SenseCAP M2 现在提供 US915 和 EU868 版本，具有 FCC 和 CE 认证。在不久的将来，它将为不同的国家和地区提供更多频段版本和相应的认证。

**特性**
============

![](https://www.sensecapmx.com/wp-content/uploads/2022/07/sensecap-m2-DO-features.png)

**· 广域覆盖和强信号**

配备 3dBi 高性能天线，SenseCAP M2 可以提供数英里的 LoRaWAN 覆盖和强信号。您可以根据不同环境的使用场景随时升级天线。

**· 卓越稳定的性能**

SenseCAP M2 采用成熟的硬件解决方案 MT7628 和 Semtech SX1302/SX1303 基带长距离芯片。它内置 BLE，支持 Wi-Fi 和以太网互联网连接。对于需要通过以太网为网关供电而不是额外电源线的用户，M2 还添加了 PoE 功能。

**· 专业管理工具和云服务**

用户可以通过 SenseCAP Hotspot App 通过几个简单步骤轻松设置热点。SenseCAP Hotspot Dashboard 和 SenseCAP Local Console 也为用户开发，以高效便捷的方式监控和管理热点。

**· 高效及时的在线支持**

除了这些专业管理工具和云服务外，SenseCAP 的技术支持一直受到客户的高度赞扬。无论何时遇到技术问题，高效的在线技术支持服务始终在这里，及时为您提供支持。

**· 安全可靠**  
对于处理大量数据传输的网关，安全性永远不会过分关注。SenseCAP M2 Data Only 始终确保高安全性身份验证和与人民网络的可靠连接。

**· 简单步骤轻松设置**

通过 SenseCAP Hotspot App 简单步骤即可使用

**· 丰富配件**

除了 SenseCAP 设备本身的强大功能外，SenseCAP 一直为用户提供丰富的配件选择，以获得热点的最大性能。例如最畅销的高性能物联网天线、天线电缆、安装垫和避雷器。

**接口**
=============

![](https://www.sensecapmx.com/wp-content/uploads/2022/07/插图-01.jpg)

**注意：**

- MicroSD 卡槽为将来使用预留。SenseCAP M2 作为 LoRaWAN 网关不需要 Micro SD 卡。
- Nano SIM 卡槽仅在 4G 版本热点中可用。设备中不包含 SIM 卡。
- Type-C 接口为高级开发者管理预留。请不要使用它为设备供电。

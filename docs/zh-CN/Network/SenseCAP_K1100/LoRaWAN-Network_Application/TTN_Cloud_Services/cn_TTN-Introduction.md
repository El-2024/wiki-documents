## 可升级为工业传感器

通过 SenseCAP [S2110 控制器](https://www.seeedstudio.com/SenseCAP-XIAO-LoRaWAN-Controller-p-5474.html) 和 [S2100 数据记录仪](https://www.seeedstudio.com/SenseCAP-S2100-LoRaWAN-Data-Logger-p-5361.html)，您可以轻松地将 Grove 转变为 LoRaWAN® 传感器。Seeed 不仅帮助您进行原型设计，还为您提供了通过 SenseCAP 系列的坚固[工业传感器](https://www.seeedstudio.com/catalogsearch/result/?q=sensecap&categories=SenseCAP&application=Temperature%2FHumidity~Soil~Gas~Light~Weather~Water~Automation~Positioning~Machine%20Learning~Voice%20Recognition&compatibility=SenseCAP)扩展项目的可能性。

IP66 外壳、蓝牙配置、与全球 LoRaWAN® 网络的兼容性、内置 19 Ah 电池以及来自 APP 的强大支持，使得 [SenseCAP S210x](https://www.seeedstudio.com/catalogsearch/result/?q=S21&categories=SenseCAP~LoRaWAN%20Device&product_module=Device) 成为工业应用的最佳选择。该系列包括土壤湿度、空气温湿度、光强、二氧化碳、电导率以及一体化 8 合 1 气象站传感器。尝试最新的 SenseCAP S210x，助力您的下一个成功的工业项目。

## 第一次使用 TTN

如果这是您第一次使用 TTN，请参考本节完成 TTN 账户的注册以及组件的使用。

### 注册 TTN 账户

进入 [TTN 网站](https://console.cloud.thethings.network//)，点击 **Sign up**。选择您偏好的计划和控制台。在这里，我们强烈建议您选择 **Community** 来完成我们的项目，因为它是免费的。

<div align="center"><img width={800} src="https://files.seeedstudio.com/wiki/Wio-Terminal-Developer-for-TTN/3.png" /></div>

:::note
自 2021 年 1 月起，The Things Network 运行 The Things Stack Community Edition，这是一个免费的、基于社区的 The Things Stack 部署。

这是对之前 The Things Network V2 的升级，带来了一系列全新的功能、开箱即用的集成、扩展的覆盖范围以及改进的用户体验。

The Things Stack Community Edition 提供了一种免费且对开发者友好的方式来熟悉 LoRaWAN® 技术。它提供了一个由 The Things Industries 发起并由来自全球 100 多个国家的 10 万多名 The Things Network 社区成员支持的公共社区 LoRaWAN® 网络。
:::

在下一页，您需要选择您的国家和地区，请根据实际情况选择。

<div align="center"><img width={800} src="https://files.seeedstudio.com/wiki/K1100/25.png" /></div>

完成账户注册后，您将进入控制台页面，可用的选项包括 **Application** 和 **Gateway**。

<div align="center"><img width={800} src="https://files.seeedstudio.com/wiki/K1100/TTN-V3-console.png" /></div>

### TTN 应用程序 - 添加设备

要接收上行消息并向终端设备发送下行消息，您需要先在 The Things Stack Community Edition 中添加设备。

设备是在 [应用程序](https://www.thethingsnetwork.org/docs/applications-and-integrations/) 中添加的。应用程序可用于按用途或位置对设备进行分组，例如。

如果使用控制台，您可以手动添加设备。

<div align="center"><img width={800} src="https://files.seeedstudio.com/wiki/K1100/19.png" /></div>

在新页面中，您需要填写应用程序 ID、应用程序名称和描述等信息。根据您的方便填写即可。

<div align="center"><img width={600} src="https://files.seeedstudio.com/wiki/K1100/21.png" /></div>

要创建设备，首先打开您希望添加设备的应用程序。在左侧菜单中转到 **End devices**，然后点击 **+ Add end device** 进入终端设备注册页面。

<div align="center"><img width={800} src="https://files.seeedstudio.com/wiki/K1100/23.png" /></div>

我们的设备包含在 LoRaWAN® 设备库中，这意味着您需要填写的信息更少！

要使用设备库，请确保选择了 **From the LoRaWAN Device Repository** 选项卡。然后，选择您的设备的 **Brand**、**Model**、**Hardware Version**、**Software Version** 和 **Region**。

<div align="center"><img width={800} src="https://files.seeedstudio.com/wiki/K1100/29.png" /></div>

:::note
选择适合您所在地区的频率计划。您的设备和网关必须使用相同的频率计划才能通信。不同的世界区域使用不同的[频率计划](https://www.thethingsindustries.com/docs/reference/frequency-plans/)，例如欧洲为 863-870 MHz，北美为 902-928 MHz 等。
:::

输入您的 **DevEUI**、**AppEUI** 和 **AppKey**。Grove - Wio-E5 是可编程的，您可以使用生成按钮生成一个 EUI，并将其编程到您的设备中。

为您的设备提供一个唯一的 **End device ID**。请参阅 [ID 和 EUI 约束](https://www.thethingsindustries.com/docs/reference/id-eui-constraints/) 以获取有关选择唯一 ID 的指南。

<div align="center"><img width={600} src="https://files.seeedstudio.com/wiki/K1100/30.png" /></div>

点击 **Register end device** 创建终端设备。

设备现在已激活，一旦发送上行消息，它将在 The Things Stack 中显示为已连接。

- 有关添加设备的更详细说明，请参阅官方 [The Things Stack 文档页面](https://www.thethingsindustries.com/docs/devices/)。
- 有关如何将 K1100 套件添加到 TTN，请参阅 [Connecting to TTN](https://wiki.seeedstudio.com/cn/Connecting-to-TTN/)。

### TTN 应用程序 - 集成

在您的应用程序中，您还可以创建集成来帮助您处理数据并通过触发事件对其进行操作。通过这种方式，您可以实现解决方案，例如使用某些云 IoT 平台监控或可视化您的传感器数据，安排命令发送到您的终端设备等。

The Things Stack Community Edition 提供了许多集成：

- [MQTT](https://www.thethingsindustries.com/docs/integrations/mqtt/) - The Things Stack Community Edition 提供了一个 MQTT 服务器，因此您可以使用 MQTT 客户端订阅来自设备的上行消息，或发布下行消息以发送到设备。
- [Webhooks](https://www.thethingsindustries.com/docs/integrations/webhooks/) - HTTP Webhooks 功能允许将与应用程序相关的消息发送到特定的 HTTP(S) 端点。如果您希望自己添加新的 Webhook 集成模板，还可以贡献到我们的 Webhook 模板开源库。
- [存储集成](https://www.thethingsindustries.com/docs/integrations/storage) - 存储集成允许您将上行消息存储在持久性数据库中，以便稍后检索。
- [AWS IoT](https://www.thethingsindustries.com/docs/integrations/aws-iot/) - The Things Stack Community Edition 允许您与 AWS IoT 集成，并将应用程序遥测数据发布到 IoT Core 端点。
- [LoRa Cloud](https://www.thethingsindustries.com/docs/reference/application-packages/lora-cloud-device-and-application-services/) - LoRa® Cloud 集成利用设备和应用服务协议，允许您在应用层管理常见功能以及基于调制解调器的设备的地理定位。

<div align="center"><img width={800} src="https://files.seeedstudio.com/wiki/K1100/26.png" /></div>

- 请查看 The Things Stack 官方文档页面，获取更多[逐步集成指南](https://www.thethingsindustries.com/docs/integrations/)。

### TTN 网关

公共社区 LoRaWAN® 网络的主要构建模块是网关。全球已有数千个网关连接到 The Things Stack Community Edition，使社区 LoRaWAN® 网络可供免费使用。

要将您的网关连接到 The Things Stack Community Edition，需要将其配置为正确地将流量转发到该平台，并且还需要在 The Things Stack Community Edition 上添加/注册网关。

建议使用 [LoRa Basics Station 协议](https://www.thethingsindustries.com/docs/gateways/lora-basics-station/)，因为 [UDP Packet Forwarder](https://www.thethingsindustries.com/docs/gateways/semtech-udp-packet-forwarder/) 存在一些安全性和可扩展性问题。<!-- - UDP Packet Forwarder链接404-->

<div align="center"><img width={800} src="https://files.seeedstudio.com/wiki/K1100/18.png" /></div>

The Things Stack 部署能够连接到 [Packet Broker](https://www.thethingsindustries.com/docs/getting-started/packet-broker/)，您可以利用 The Things Network 的覆盖范围，而无需添加任何自己的网关。请参阅[添加网关](https://www.thethingsindustries.com/docs/gateways/adding-gateways/)以获取添加网关的说明，并了解您的位置是否有附近的网关。

<div align="center"><img width={800} src="https://files.seeedstudio.com/wiki/K1100/overview.png" /></div>

尽管您的区域可能已经有[覆盖范围](https://www.thethingsnetwork.org/map)，我们仍然建议您获取一个网关以确保可靠的覆盖范围。市场上有许多功能丰富的网关，请确保选择最适合您需求的网关。

- 如果在将网关添加到 The Things Stack Community Edition 时遇到问题，请参阅[故障排除](https://www.thethingsindustries.com/docs/gateways/adding-gateways/troubleshooting/)。

有关在 TTN 中使用和配置网关的更多信息，请阅读网关的说明书或向您购买网关的供应商寻求帮助。

:::note
不同的世界区域使用不同的[频率计划](https://www.thethingsindustries.com/docs/reference/frequency-plans/)，例如欧洲使用 863-870 MHz，北美使用 902-928 MHz 等。在购买之前，请务必仔细检查硬件是否与您所在地区的无线电频率一致！
:::

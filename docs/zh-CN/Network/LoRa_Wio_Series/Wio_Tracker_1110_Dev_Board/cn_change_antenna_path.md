Wio Tracker 1110 开发板默认使用板载 LoRa 天线，为了获得更好的信号，我们可能需要使用外部天线。本章节将指导用户如何更改天线路径以连接外部 LoRa 天线。

* 移除 `R3`。
* 将 `R4` 更改为 0R。

![](https://files.seeedstudio.com/wiki/Wio_Link_Deluxe_Kit/img/Wio_Link_Deluxe_Kit_product_view_1200_s.jpg)

Este kit es un paquete completo de todos los módulos funcionales generales requeridos para alguien que pretende hacer algún proyecto real o crear un prototipo. Contiene sensores (principalmente), módulos de función de conmutación, servomotores, tira LED y un altavoz. La parte más increíble de este kit es la placa de desarrollo Wio Link que puede ser utilizada para varios dispositivos: sin requerir ninguna habilidad de programación. Todas las acciones que quieras que tus módulos realicen y la operación de flasheo de código al Wio Link pueden completarse con toques en una aplicación móvil. Y puedes enviar instrucciones a tus dispositivos a través de internet desde cualquier lugar. Así que Wio Link será una utilidad perfecta para hacer las cosas alrededor más inteligentes para todos. ¡Ahora hagamos algo real!

[![](https://files.seeedstudio.com/wiki/common/Get_One_Now_Banner.png)](https://www.seeedstudio.com/Wio-Link-Deluxe-Kit-p-2615.html)

:::caution
     La función Wio IFTTT está descontinuada. Pero la APP Wio está disponible. Puedes usar la API de la APP Wio para leer el estado del sensor y controlar el actuador. 
:::
Características
--------

-   Paquete completo de todos los módulos funcionales generales requeridos
-   Interfaz de puerto Grove, función más práctica y menos trabajo
-   Completamente fácil de hacer las cosas a tu alrededor más inteligentes y agregar diversión a la vida diaria
-   Costo-efectivo
-   Peso: 305 g

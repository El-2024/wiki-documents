# Ejemplos de API TFT LCD

Este repositorio proporciona algunos gifs para los ejemplos de la biblioteca TFT LCD, todos los ejemplos se pueden encontrar en `Archivo`->`Ejemplo`->`Seeed_Arduino_LCD` -> `320 x 240`. Puedes usar estos ejemplos para implementar en tu diseño.

Para más información sobre la pantalla TFT LCD, por favor visita [Fundamentos de la Pantalla TFT LCD](https://wiki.seeedstudio.com/es/Wio-Terminal-LCD-Basic/).

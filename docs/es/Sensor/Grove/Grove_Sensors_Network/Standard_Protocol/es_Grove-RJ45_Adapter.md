El Adaptador Grove-RJ45 es un módulo que puede convertir un conector Grove estándar a un conector RJ45. Los cables Grove universales de 4 pines no pueden cumplir con los requisitos cuando Arduino necesita conectar y controlar remotamente un módulo Grove. A través de dos adaptadores RJ45 y un cable, es posible lograr comunicación entre Arduino y el módulo Grove. Esto ayuda a resolver el problema de que Arduino y Grove no son convenientes de conectar debido a problemas de distancia.

[![](https://files.seeedstudio.com/wiki/Seeed-WiKi/docs/images/300px-Get_One_Now_Banner-ragular.png)](https://www.seeedstudio.com/Grove-RJ45-Adapter-p-1426.html)

## Características

---

* Conector Grove estándar y conector RJ45

* Aplicaciones amplias

* Fácil de usar

## Uso

---
Si quieres usar el Sensor de Humedad para detectar en tiempo real la humedad del suelo en tu jardín, pero no quieres ir al jardín personalmente. ¿Cómo hacerlo? Usando dos adaptadores RJ45 y un cable de red largo, puedes resolver este problema.
La instalación del hardware debería ser:

![](https://files.seeedstudio.com/wiki/Grove-RJ45_Adapter/img/RJ45.jpg)

Puede que necesites escribir el código por ti mismo para aplicaciones especiales. Pero podemos garantizar que siempre puedes conocer la humedad del suelo de las plantas observando los datos de salida del sensor de humedad.

<font color="red">Puede que quieras preguntar por qué no usar un cable Grove largo en lugar de dos Grove - RJ45 y un cable de red, la razón es que el cable de red tiene una atenuación débil.</font>

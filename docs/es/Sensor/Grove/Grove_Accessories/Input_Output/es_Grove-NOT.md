Este módulo Grove proporciona la función de una [puerta lógica NOT](https://en.wikipedia.org/wiki/Inverter_%28logic_gate%29). Está basado en el chip 74HC1G04GW. El 74HC1G04GW es un dispositivo CMOS de alta velocidad con puerta Si que proporciona el buffer inversor. El módulo NOT tiene una entrada en el lado izquierdo y una salida en el lado derecho. Cuando la señal de entrada es HIGH, la señal de salida es LOW. Cuando la señal de entrada es LOW, la señal de salida es HIGH.

[![](https://files.seeedstudio.com/wiki/Seeed-WiKi/docs/images/300px-Get_One_Now_Banner-ragular.png)](https://www.seeedstudio.com/Mixer-Pack-V2-(Electronic-blocks%2Cwithout-Arduino%2Cplug-and-play-system)-p-1867.html)

##  Características

*   Función simple de puerta NOT con una entrada y una salida

*   Hace que integrar una puerta lógica NOT en tu circuito sea muy fácil

*   Utiliza cables Grove estándar de 4 pines para conectar con otros módulos Grove

##  Uso

*   Consulta el ejemplo de Luces LED que Brillan en la Oscuridad en la página wiki del Grove Mixer Pack V2 para ver cómo construir un circuito que utiliza lógica NOT.

*   También consulta el uso típico del módulo Grove - NOT

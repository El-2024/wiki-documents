I2C Hub Grove es un módulo Grove de extensión para conectar múltiples dispositivos I2C al Grove Base Shield. Se puede usar con [Cable Universal de 4 Pines a X2 4 Pines](https://www.seeedstudio.com/depot/universal-4-pin-to-x2-4-pin-cable-5-pcs-pack-p-847.html?cPath=178_179) y conecta hasta 3 dispositivos I2C, lo que puede cubrir la mayoría de propósitos de desarrollo.

[![](https://files.seeedstudio.com/wiki/common/Get_One_Now_Banner.png)](https://www.seeedstudio.com/Grove-I2C-Hub-p-851.html)

Características
--------

- Encadenable

:::caution
    Las plataformas mencionadas anteriormente como soportadas son una indicación de la compatibilidad de software o teórica del módulo. Solo proporcionamos biblioteca de software o ejemplos de código para la plataforma Arduino en la mayoría de los casos. No es posible proporcionar biblioteca de software / código de demostración para todas las plataformas MCU posibles. Por lo tanto, los usuarios tienen que escribir su propia biblioteca de software.
:::

Ideas de Aplicación
-----------------

- Usar más dispositivos I2C de los que normalmente tendrías espacio en tu plataforma Grove.

Uso
-----

No hay conflicto si usas el mismo Socket I2C simultáneamente porque cada dispositivo I2C tiene su propia dirección. La instalación de hardware se muestra a continuación.

![](https://files.seeedstudio.com/wiki/Grove-I2C_Hub/img/I2C_Hub_Connect.jpg)

Aquí no listamos un ejemplo específico.

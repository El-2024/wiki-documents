Grove - Receptor FM es un módulo receptor FM de banda ancha, este módulo está basado en SX6119, ofrece un verdadero receptor FM de voz de chip único, que tiene amplificador integrado, VCO, filtro y demodulador. El botón multifunción proporciona mayor comodidad para que los usuarios controlen.

[![](https://files.seeedstudio.com/wiki/common/Get_One_Now_Banner.png)](https://www.seeedstudio.com/Grove-FM-Receiver-p-1841.html)

Características
--------

- Interfaz Grove
- Receptor FM con un rango de sintonización de 64 MHz a 108 MHz cubriendo las bandas de Europa del Este (OIRT), Japón, Europa y Estados Unidos
- Menor consumo de energía
- Botón multifunción
- Indicador LED
- Interfaz para auriculares

:::tip
Más detalles sobre los módulos Grove consulte [Sistema Grove](https://wiki.seeedstudio.com/es/Grove_System/)
:::

Descripción del Hardware
------------------

![](https://files.seeedstudio.com/wiki/Grove-FM_Receiver/img/Grove-FM_v2.0_Top.jpg)

- ①：Interfaz para auriculares: Puede manejar auriculares o audio de 16 ohm o 32 ohm.
- ②：Botón multifunción: Cambiar volumen y seleccionar canal

  - Centro：Abrir FM/Cerrar FM（Mantener presionado 1S）
  - Arriba：aumentar volumen
  - Abajo：disminuir volumen
  - Izquierda：búsqueda hacia abajo
  - Derecha：búsqueda hacia arriba

- ③：Interfaz Grove
- ④：Indicador LED

  - FM Cerrado：LED apagado
  - FM Abierto：LED encendido
  - FM Buscando：LED parpadeando

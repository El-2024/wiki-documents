Grove - I2C Hub (6 Port) es un módulo de extensión Grove para conectar múltiples dispositivos I2C al Grove Base Shield. Se puede usar con el [cable Universal 4 Pin to X2 4 Pin](https://www.seeedstudio.com/depot/universal-4-pin-to-x2-4-pin-cable-5-pcs-pack-p-847.html?cPath=178_179) y conecta hasta 5 dispositivos I2C, lo que puede cubrir la mayoría de propósitos de desarrollo. Grove - I2C Hub (6 Port) tiene dos pines más que Grove - I2C Hub. Por lo tanto, Grove - I2C Hub (6 Port) satisface necesidades más diversificadas.

<p style={{textAlign: 'center'}}><a href="https://www.seeedstudio.com/Grove-I2C-Hub-6-Port-p-4349.html" target="_blank"><img src="https://files.seeedstudio.com/wiki/wiki_english/docs/images/get_one_now_small.png" width={200} height={38} border={0} /></a></p>

## Características

- Encadenable
- Dos conectores más que [Grove - I2C Hub](https://www.seeedstudio.com/Grove-I2C-Hub.html).
- Económico y práctico

Ideas de Aplicación
-----------------

- Usar más dispositivos I2C de los que normalmente tendrías espacio en tu plataforma Grove y dos conectores más que Grove I2C Hub.

Uso
-----

Es lo mismo que con Grove I2C Hub, no hay conflicto si usas el mismo Socket I2C simultáneamente porque cada dispositivo I2C tiene su propia dirección. La instalación de hardware se muestra a continuación.

Grove - Terminal de Tornillo con cuatro pines de paso de 3.5mm clasificados hasta 125V @ 6A. El terminal puede aceptar cable de 30 a 20AWG. Es el terminal de tornillo de paso más común. Este grove tiene cuatro puertos, incluyendo GND, VCC
y otros dos puertos no definidos. Los terminales de tornillo se usan comúnmente para conectar una tierra de chasis, como en un tocadiscos o un protector contra sobretensiones. La mayoría de los sistemas de megafonía en edificios también los usan para altavoces, y a veces para otras salidas y entradas.

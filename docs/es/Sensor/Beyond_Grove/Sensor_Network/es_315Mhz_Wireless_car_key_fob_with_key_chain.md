Este es un llavero inalámbrico para coche con 4 botones: Bloquear, desbloquear, silenciar y activar alarma. Un LED azul se encenderá cuando se presione cualquier botón. Usa este llavero para comunicarte con el [Relé Inalámbrico Codec-Adaptativo](https://www.seeedstudio.com/depot/codecadaptive-wireless-relay-p-550.html) para tu proyecto de control remoto.

[![](https://files.seeedstudio.com/wiki/Seeed-WiKi/docs/images/300px-Get_One_Now_Banner-ragular.png)](https://www.seeedstudio.com/depot/315mhz-wireless-car-key-fob-with-key-chain-battery-included-p-557.html?cPath=139_140)

##   Características
---
*   Rango de frecuencia: Alto: 315.15MHZ  Promedio: 315.025MHZ  Bajo: 314.85MHZ

*   Esquemas de modulación: modulación de amplitud (AM)

*   Velocidad de transferencia de datos: 625HZ

*   Potencia de salida: ≤ 12mW

*   Voltaje de batería: DC12V

*   Temperatura de funcionamiento: -30 ℃ ~ +85 ° C

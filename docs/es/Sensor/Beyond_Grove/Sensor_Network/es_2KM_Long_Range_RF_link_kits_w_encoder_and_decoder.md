![](https://files.seeedstudio.com/wiki/2KM_Long_Range_RF_link_kits_w_encoder_and_decoder/img/2kmrf.jpg)

Este es un kit de enlace RF de 433mhz de súper largo alcance. Viene con tecnología VCO, PLL, frecuencia estable y súper capacidad anti-interferencia. Puedes usarlo directamente con tus proyectos, como transporte de datos inalámbrico, control remoto, etc.

[![](https://files.seeedstudio.com/wiki/Seeed-WiKi/docs/images/300px-Get_One_Now_Banner-ragular.png)](https://www.seeedstudio.com/2km-long-range-rf-link-kits-w-encoder-and-decoder-p-321.html?cPath=139_140)

##   Especificaciones
---
| Parámetro              | Valor                                    |
|------------------------|------------------------------------------|
| Voltaje de trabajo     | Receptor(3-5V), Transmisor(3-9V)        |
| Corriente de trabajo   | ≤2.5mA(5.0VDC)                          |
| Principio de operación | Superheterodino（VCO, PLL）              |
| Modulación             | OOK/ASK                                  |
| Banda de trabajo       | 433.92MHz (servicio personalizado disponible) |
| Ancho de banda         | 1.5MHz                                   |
| Sensibilidad           | -105dBm (50Ω)                            |
| Velocidad              | ＜5Kbps                                  |
| Forma de decodificación| PT2272                                   |
| Longitud de antena     | 18cm                                     |
| Rango de transmisión   | 2 km                                     |
##   Ideas de Aplicación
---
*   Interruptor inalámbrico
*   Transmisión remota
*   Control inalámbrico

## Primeros Pasos
---

- Por favor sigue las [Instrucciones](https://files.seeedstudio.com/wiki/2KM_Long_Range_RF_link_kits_w_encoder_and_decoder/res/2KM_RF.rar) para configurar el sistema de comunicación.
- Conecta la batería, botón y placas de transmisión.
![](https://files.seeedstudio.com/wiki/2KM_Long_Range_RF_link_kits_w_encoder_and_decoder/img/2KM_TX.JPG)

!!!Note
    ¡Por favor usa una fuente de alimentación de 3-5V! 9V dañará la placa receptora.
- Conecta la batería, LEDs y placas receptoras.
![](https://files.seeedstudio.com/wiki/2KM_Long_Range_RF_link_kits_w_encoder_and_decoder/img/2KM_RX.JPG)

!!!Note
    No pongas el módulo RX y el módulo TX demasiado cerca: esto evitaría que funcionen. Asegúrate de que el módulo RX y el módulo TX estén al menos a 1 metro de distancia entre sí.
- Cuando presionamos el botón en el lado del Transmisor, el LED relacionado en el lado del receptor se encenderá.

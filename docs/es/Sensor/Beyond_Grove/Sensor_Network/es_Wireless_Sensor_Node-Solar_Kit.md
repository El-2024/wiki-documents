**Nodo de Sensor Inalámbrico - Kit Solar** es un paquete fácil de usar creado para experimentar con XBee y módulos inalámbricos independientes compatibles con **XBee** como RFBee y [Wifi Bee](/Wifi_Bee "Wifi Bee"). Todo **maker** se enfrenta a la pregunta de seleccionar el conjunto correcto de componentes para construir un **Nodo de Sensor Inalámbrico**. **Nodo de Sensor Inalámbrico - Kit Solar** llena este lugar muy apropiadamente. El Grove - XBee Carrier proporciona fuente de alimentación LDO, cargador para batería LiPo y puerto de programación para módulos compatibles con **XBee**. El **Panel Solar** proporciona el voltaje de carga requerido en áreas remotas. La **Batería LiPo de 500 mAH** proporciona el respaldo cuando el Sol no está presente. Un módulo inalámbrico correctamente diseñado y configurado podría proporcionar muchas horas de uso. Para esto, pon el nodo en modo de suspensión cuando no esté transmitiendo el valor del sensor. Este kit viene con una **Caja Transparente** perfectamente hecha, conjuntos de **Tornillos** y **Remaches de Plástico**.

- Te sugerimos comprar más de un kit para crear una matriz distribuida de Nodos de Sensor.

- Bees Shield junto con **Seeeduino** y **Wifi Bee** puede actuar como Puerta de Enlace de Internet Inalámbrica para estos nodos.

[![](https://files.seeedstudio.com/wiki/Seeed-WiKi/docs/images/300px-Get_One_Now_Banner-ragular.png)](https://www.seeedstudio.com/Wireless-Sensor-Node-Solar-Kit-p-919.html)

### Guía de Inicio Rápido

Esta sección proporciona una construcción paso a paso de un Nodo de Sensor Inalámbrico. Las imágenes son solo para ilustrar el procedimiento de ese paso. Hemos usado dos configuraciones diferentes de Grove - Xbee Carrier y módulos Grove. Así que, simplemente sigue el procedimiento.

| Paso | Procedimiento                                                                                                        | Ilustración                                                                                                                                 |
|------|------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------|
| 1    | Abre la caja como se muestra                                                                                            | ![](https://files.seeedstudio.com/wiki/Wireless_Sensor_Node-Solar_Kit/img/Wireless_Sensor_Node-Solar_Kit_Top_PlaceForOpening.jpg)   |
| 1.1  | Coloca la Batería LiPo dentro de la Caja                                                                            | ![](https://files.seeedstudio.com/wiki/Wireless_Sensor_Node-Solar_Kit/img/Wireless_Sensor_Node-Solar_Kit_Placing_Battery.jpg)       |
| 1.2  | Inserta el Conector del Panel Solar                                                                                 | ![](https://files.seeedstudio.com/wiki/Wireless_Sensor_Node-Solar_Kit/img/Wireless_Sensor_Node-Solar_Kit_Construction1.jpg)         |
| 1.3  | Abre el remache de plástico                                                                                           | ![](https://files.seeedstudio.com/wiki/Wireless_Sensor_Node-Solar_Kit/img/Wireless_Sensor_Node-Solar_Kit_Box_Plastic_Rivet1.jpg)    |
| 1.4  | Un remache abierto se ve así                                                                                  | ![](https://files.seeedstudio.com/wiki/Wireless_Sensor_Node-Solar_Kit/img/Wireless_Sensor_Node-Solar_Kit_Box_Plastic_Rivet2.jpg)    |
| 1.5  | Conecta un módulo Grove con la ayuda de Remaches. Mantén el Grove - Bee Carrier como se muestra dentro de la caja transparente. | ![](https://files.seeedstudio.com/wiki/Wireless_Sensor_Node-Solar_Kit/img/Wireless_Sensor_Node-Solar_Kit_Construction2.jpg)         |
| 2.3  | Un remache completamente cerrado se ve así                                                                             | ![](https://files.seeedstudio.com/wiki/Wireless_Sensor_Node-Solar_Kit/img/Wireless_Sensor_Node-Solar_Kit_Rivet_Bottom.jpg)          |
| 3.0  | Conecta el Cable Grove de 4 Hilos, la Batería y los Conectores del Panel Solar al Grove - XBee Carrier                        | ![](https://files.seeedstudio.com/wiki/Wireless_Sensor_Node-Solar_Kit/img/Wireless_Sensor_Node-Solar_Kit_TwigConnection.jpg)        |
| 4.0  | Inserta el módulo XBee. Fija el Tornillo en las cuatro esquinas                                                                | ![](https://files.seeedstudio.com/wiki/Wireless_Sensor_Node-Solar_Kit/img/Wireless_Sensor_Node-Solar_Kit_Fix_Screw.jpg)             |
| 5.0  | Cierra la tapa. El puerto USB y el Interruptor de Encendido son accesibles desde el lateral                                          | ![](https://files.seeedstudio.com/wiki/Wireless_Sensor_Node-Solar_Kit/img/Wireless_Sensor_Node-Solar_Kit_Wires_Switch_USB_Side.jpg) |
| 6.0  | Conecta el cable USB y programa el XBee                                                                       | ![](https://files.seeedstudio.com/wiki/Wireless_Sensor_Node-Solar_Kit/img/Wireless_Sensor_Node-Solar_Kit_USBProgramming.jpg)        |

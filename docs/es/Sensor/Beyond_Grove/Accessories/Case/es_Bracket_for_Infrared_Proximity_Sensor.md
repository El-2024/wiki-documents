Este es un kit de soporte de aluminio para sensor de proximidad infrarrojo para Sensor de Proximidad Infrarrojo. Con este kit, un Servo RB-421 y un Sensor de Proximidad Infrarrojo puedes construir un ensamble de montaje servo de 2 grados de libertad.
Este kit es profesional para dirigir el sensor de proximidad infrarrojo en un robot para 180° de rotación de lado a lado y de adelante hacia atrás.
Estas partes también tienen agujeros con espaciado de 20mm para que puedan ser conectadas al Arnés de Paquete Inicial de Seeed (https://www.seeedstudio.com/depot/starter-bundle-harness-p-906.html?cPath=178_184) que también tiene espaciado de 20mm.

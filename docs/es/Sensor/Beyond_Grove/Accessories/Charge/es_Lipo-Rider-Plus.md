El Lipo Rider Plus de Seeed no es solo un Cargador Rápido sino también un Elevador de Potencia Fuerte, soporta hasta 5V/2A de corriente de carga, integración de salida elevadora, salida de hasta 5V/2.4A, también puede dar salida de 3.3V/250mA. La salida de 3.3V está normalmente encendida, y la salida de 5V soporta control IO o interruptor manual, lo que significa que puedes usar 3.3V para suministrar energía a un microcontrolador y usar la E/S de este microcontrolador para controlar la salida de 5V, totalmente programable. ¡Wow, no puedo creer que cueste menos de $5!

En pocas palabras, el Lipo Rider Plus puede cargar la batería Lipo mucho más rápido, al mismo tiempo puede proporcionar energía fuerte para el dispositivo esclavo. Oops, olvidé introducir la diferencia entre el Lipo rider y el cargador Lipo tradicional.

## Características

- Hasta 5V / 2A de corriente de carga - USB Type C
- Hasta 5V / 2.4A de corriente de salida - USB Type A
- Proporciona salida de 3.3V / 250mA - Conector de Pines
- Soporte para salida de apagado (control IO/interruptor manual)
- LED indicador de estado de carga integrado + LED indicador de energía de 4 segmentos integrado
- Ultra-pequeño 2.5cm*4.1cm
- Diseño de una sola capa fácil de montar

### Lipo Rider VS Cargador Lipo

Como sabemos, el Cargador Lipo tradicional puede cargar la batería de litio. Bueno, usualmente solo pueden cargar. Pero ¿qué puede hacer un Lipo Rider?

No solo un cargador sino también una fuente de alimentación y un elevador. Con la ayuda del Lipo Rider Plus, puedes ingresar una batería lipo de 3.7V, y dar salida de hasta 5V/2.4A a través del puerto USB Type-A. Con el interruptor integrado, puedes controlar el ENCENDIDO/APAGADO del puerto USB fácilmente. Mientras tanto, proporcionamos el conector de pines de función completa, que también puede dar salida de 3.3V/250mA.

### Lipo Rider Plus VS Lipo Rider & Lipo Rider Pro

Puedes encontrar que ya hemos lanzado el [Lipo Rider V1.3](https://www.seeedstudio.com/Lipo-Rider-v1-3.html) y [Lipo Rider Pro](https://www.seeedstudio.com/LiPo-Rider-Pro-p-992.html), entonces, ¿cuál es la diferencia entre [Seeed Lipo Rider](https://www.seeedstudio.com/tag/Lipo-Rider.html)? En resumen, tanto LiPo Rider Pro como Lipo Rider v1.3 tienen dos Puertos JST2.0, lo que te permite usar un panel solar para cargar una batería lipo.

Sin embargo, el Lipo Rider Plus solo tiene un puerto JST2.0. Por lo tanto, no puede usar un panel solar como fuente de energía. La buena noticia es que Lipo Rider Plus se ha vuelto más compacto, es casi la mitad del tamaño de LiPo Rider Pro y Lipo Rider v1.3, puedes montarlo fácilmente en cualquier proyecto. Más importante aún, su rendimiento es mucho más fuerte que sus dos predecesores. En resumen, si necesitas trabajar con paneles solares, entonces LiPo Rider Pro y Lipo Rider v1.3 son adecuados para ti; si quieres experimentar carga rápida, entonces Lipo Rider Plus será tu mejor opción.

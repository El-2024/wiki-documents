Este es un panel solar personalizado, que se conecta directamente con muchas de nuestras placas de desarrollo y tiene una alta eficiencia del 15%. La unidad tiene un recubrimiento epoxi transparente con respaldo de tablero duro. ¡Sellado robusto para aplicaciones exteriores!

[![](https://files.seeedstudio.com/wiki/Seeed-WiKi/docs/images/300px-Get_One_Now_Banner-ragular.png)](https://www.seeedstudio.com/2.5W-Solar-Panel-116X160-p-953.html)

##   Especificaciones
---
*   Tamaño PCB : 116*160*1.5 mm

*   Monolítico : 9.57*156 mm

*   Eficiencia : 15%

*   Voltaje:5.5V;

*   Corriente:450mA

*   Potencia:2.5W
*   Conector: 2.0mm JST

*   Tipo:156 Dos líneas;

*   Material PCB : toda fibra de vidrio

*   Requisitos PCB : positivo

*   Requisitos de calidad :

    *   planitud menor a 0.1mm
    *   limpio, no se permite tener rayones
    *   fenómeno de desprendimiento de pintura, tal como un agujero no se permite tener desviación

*   **Nota** : tolerancia Positiva y negativa 0.1mm

*   Nombre: laminado PET (116*160)

*   Disposición:1*11(serie) SIN espaciado pegado artical negro;

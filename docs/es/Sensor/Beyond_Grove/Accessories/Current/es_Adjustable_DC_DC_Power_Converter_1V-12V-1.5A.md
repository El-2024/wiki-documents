![](https://files.seeedstudio.com/wiki/Adjustable_DC_DC_Power_Converter_1V-12V-1.5A/img/Adjustable_04.jpg)

Este es un regulador de voltaje reductor DC/DC que convierte voltaje de entrada entre 4.5V y 16V en un voltaje menor entre 1V y 12V, capaz de manejar una carga de 1.5A con excelente regulación de línea y carga.

En algunos proyectos, podrías necesitar diferentes voltajes (1-12V) para tus dispositivos. Sin este tipo de fuente de alimentación DC, encontrarás que es muy tardado comprar una nueva. El retraso del proyecto es muy molesto. ¡No te preocupes! Este módulo puede acelerar el prototipado, porque puedes obtener el voltaje que necesitas (como 1.2V, 1.8V, 3.3V, 5V y 9V) tan fácil como parpadear.

## Especificaciones

---

* Voltaje de Entrada: 4.5-16V
* Voltaje de Salida: 1V - 12VDC
* Consumo de Calentamiento: 0.5-1200mW
* Resistencia de Carga: ajustable
* Corriente Máxima de Salida: 1.5A
* Rizado de Salida: ≤100mV
* Frecuencia de Conmutación: 500KHz
* Temperatura de Operación: -40℃ a +125℃
* Dimensiones: 36mm*30mm*10mm(l*a*h)

## Características

---

* Amplio Rango de Voltaje de Salida
* Corriente de Carga de Salida Garantizada de 1.5A
* Rango de Voltaje de Entrada Hasta 16V
* Excelente Regulación de Línea y Carga
* Modo de Espera de Bajo Consumo
* Potenciómetro preciso para ajuste de voltaje
* Protección contra Sobrecalentamiento y Sobrecorriente
* Alimentación para protoboard

## Demostración

---
Conecta el módulo con la Protoboard como se muestra en la siguiente imagen. Puede seleccionar rápidamente 3.3V, 5V o una salida de voltaje ajustable, a través del puente. Cuando se selecciona el ajustable, puede ajustarse mediante R12, para ajustar el rango de voltaje de salida (1-12V). Al mismo tiempo, puedes usar Arduino para leer el voltaje de salida A0, para obtener el valor de voltaje preciso.

Este es un medidor de capacitancia simple pero muy útil. La nueva versión corrigió errores de diseño de la versión anterior y vino con varias mejoras:

*   Diseño de PCB mejorado.
*   Se agregó conector DC popular para fácil conexión de la fuente de alimentación.
*   Se agregó interruptor. Esto es deseable cuando funciona con batería.
*   Se agregaron orificios de montaje.
*   Polaridad marcada.
*   Tamaño más pequeño.

[![](https://files.seeedstudio.com/wiki/Seeed-WiKi/docs/images/300px-Get_One_Now_Banner-ragular.png)](https://www.seeedstudio.com/Capacitance-Meter-Kit-p-268.html)

##   Características

*   Aproximadamente 1% de precisión
*   Rango de medición: 1pF - 500uF
*   Cambio automático de rango
*   Puesta a cero disponible
*   Salida serie en tiempo real de las lecturas de medición con marca de tiempo
*   Bajo costo y fácil de construir


##   Lista de Materiales (BOM) / lista de partes

**El Kit Incluye:**

*   Todas las partes excluyendo la fuente de alimentación
*   Esquemático
*   Manual

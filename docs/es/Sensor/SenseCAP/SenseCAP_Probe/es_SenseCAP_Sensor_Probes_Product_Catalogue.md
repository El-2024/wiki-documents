## Acerca de SenseCAP

Entre el primer lanzamiento de la serie de productos IIoT de Seeed, SenseCAP se enfoca en aplicaciones de detección ambiental inalámbrica: agricultura inteligente, agricultura de precisión y ciudad inteligente, por nombrar algunas. Consiste en productos de hardware (sensores, registradores de datos y gateways, etc.), servicios de software (SenseCAP Mate App, SenseCAP Web Portal, panel de control abierto) y API para gestión de dispositivos y datos.

## Acerca de las Sondas de Sensores

Nuestra gama de sondas de sensores industriales SenseCAP están diseñadas para resistir entornos hostiles como vibraciones y exposición a químicos y gases, lo cual es común en aplicaciones industriales, y nuestra gama industrial de sensores está equipada con clasificaciones IP serias desde IP66 hasta IP68 para asegurar que los productos funcionen correctamente en sus respectivos entornos hostiles.
Las Sondas de Sensores SenseCAP también cuentan con conexiones de proceso para asegurar un ajuste seguro en el medio que se está midiendo, junto con transmisores de datos de sensores que convierten la señal para adaptarse a sistemas de instrumentación industrial.
Algunos de los Sensores Industriales MODBUS RS485 vienen en dos versiones. La principal diferencia es el conector del cable. Por favor, elija la versión que mejor se adapte a sus necesidades:
（1） El que tiene cables de conexión se sugiere usar con [SenseCAP LoRaWAN® Data Logger](https://www.seeedstudio.com/SenseCAP-S2100-LoRaWAN-Data-Logger-p-5361.html?queryID=82d7de6ca64b8316686bdfe2c05664d2&objectID=5361&indexName=bazaar_retailer_products) para conectar con la red LoRaWAN®, disfrutando los beneficios de comunicación de bajo consumo y largo alcance.
（2）El que tiene conector de aviación ya preconfigurado [SenseCAP Sensor Hub 4G Data Logger](https://www.seeedstudio.com/SenseCAP-Sensor-Hub-4G-Data-Logger-with-builtin-battery-p-4852.html?queryID=b8ef1fa7e141765c30033fc3d8eddf10&objectID=4852&indexName=bazaar_retailer_products) con experiencia plug-and-play para recopilar varios datos de sensores vía 3G/4G LTE.

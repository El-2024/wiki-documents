**Los pasos específicos para subir el código:**

1. Selecciona el tipo de placa Arduino que estás usando por la ruta: Tools --&gt; Board --&gt; por ejemplo, Arduino UNO después de haber abierto una demostración.
![](https://files.seeedstudio.com/wiki/Upload_Code/img/Open_code.jpg)

Puede ser tu elección incorrecta del tipo de placa si aparece la imagen de abajo, por favor vuelve a seleccionar la placa Arduino que estás usando.

![](https://files.seeedstudio.com/wiki/Upload_Code/img/Error_score.jpg)

2. Selecciona el puerto serie correcto que estás usando por la ruta: Tools --&gt; Serial Port --&gt; por ejemplo, COM3.
3. Sube el código de demostración como se muestra a continuación:

![](https://files.seeedstudio.com/wiki/Upload_Code/img/Upload_state.jpg)

Cuando aparezca "Done uploading", felicidades, has subido exitosamente.

El Kit Avanzado Sidekick incluye un conjunto de componentes básicos así como muchos sensores y otros módulos.

Tenemos un libro de bocetos para el capítulo de Sensores. puedes descargarlo [aquí](https://github.com/Seeed-Studio/sketch_sidekick).

Cuando hayas descargado el código, extráelo a la carpeta F:, o donde prefieras.

Luego abre Arduino IDE, haz clic en Archivo -&gt; Preferencias:

![](https://files.seeedstudio.com/wiki/Sketchbook_of_Sidekick_Advanced_Kit/img/Sidekick_40_1.png)

Pon la ubicación del Libro de bocetos en el código que habías descargado, haz clic en OK para terminarlo. Vuelve a abrir Arduino IDE.

Haz clic en Archivo -&gt; Libro de bocetos, entonces puedes encontrar algunos proyectos ahora, ese es el código de demostración de este capítulo.

![](https://files.seeedstudio.com/wiki/Sketchbook_of_Sidekick_Advanced_Kit/img/Sidekick_40_2.png)

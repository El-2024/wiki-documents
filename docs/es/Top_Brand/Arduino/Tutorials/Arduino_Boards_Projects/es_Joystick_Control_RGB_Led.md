Puedes usar un Joystick en algunas ocasiones de control, como controlar un coche de juguete.

Esta demostración te mostrará cómo usar un joystick, y necesitaremos también un LED RGB, cuando el joystick esté en diferentes posiciones, el LED RGB mostrará diferentes colores.

**Hardware Requerido：**

* Arduino

* Joystick

* LED RGB

**Circuito：**

El Joystick se conecta a A0, A1, el LED RGB se conecta a D3, D9 y D10.

![](https://files.seeedstudio.com/wiki/Joystick_Control_RGB_Led/img/Sidekick_38_1.png)

**Código:**

Abre Arduino IDE, haz clic en File -&gt; Sketchbook -&gt; joystick.

El Motor Shield es un módulo controlador para motores que te permite usar Arduino para controlar la velocidad de trabajo y la dirección del motor

|![](https://files.seeedstudio.com/wiki/Motor_Shield/img/Smotoshield2.jpg)|![](https://files.seeedstudio.com/wiki/Motor_Shield/img/Motorshield_01.jpg)|
|---|---|
|[Motor Shield V1.0](https://seeeddoc.github.io/Motor_Shield_V1.0/)|[Motor Shield V2.0](https://seeeddoc.github.io/Motor_Shield_V2.0/)|

Aquí se lista la comparación entre varias versiones del Motor Shield:

 |Parámetro|Motor Shield V1.0|Motor Shield V2.0|
 |---|---|---|
|**Voltaje de Trabajo**|+5V|+5V|
 |**Alimentación**|Fuente de alimentación externa|Alimentación externa o Pin +5 de Arduino/Seeeduino|

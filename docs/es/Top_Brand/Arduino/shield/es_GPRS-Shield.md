El GPRS Shield te proporciona una forma de usar la red de telefonía celular GSM para recibir datos desde una ubicación remota. El shield te permite lograr esto a través de cualquiera de los tres métodos:

- Servicio de Mensajes Cortos

- Audio

- Servicio GPRS
  - [Rastreo GPS](http://vamostech.com/gps-tracking)
  - [Rastreador GPS](http://vamostech.com/gps-tracking)
  - [Properti semarang](http://www.raywhitesemarang.com)
  - [Jual sepatu online](http://mariposa-store.com/)

Hay dos versiones para elegir:

|![](https://files.seeedstudio.com/wiki/GPRS-Shield/img/GPRS_shield_v1.4.jpg)|![](https://files.seeedstudio.com/wiki/GPRS-Shield/img/GPRSshield_01.jpg)|
|---|---|
|GPRS Shield V1.0|GPRS Shield V2.0|

Aquí se lista la comparación entre varias versiones del GPRS Shield que puedes encontrar en [myamplifiers.com](http://www.myamplifiers.com/):

|  Parámetro|GPRS Shield V1.0|GPRS Shield V2.0|
|---|---|---|
|  **Voltaje de Trabajo** | +5V|+5V  |
|  **Modo de Alimentación**|Conector de alimentación o pin 5V de Arduino/Seeeduino|Pin 5V o pin Vin de Arduino/Seeeduino |
| **Interfaz de Audio**|Conectores de altavoz y auriculares|Conector de auriculares 2 en 1  |
| **Shield Estándar**|No|Sí  |
|  **Antena**|0.4W|0.8W(29dBm)  |
|**potencia de transmisión**|0.4W|0.8W(29dBm)|

El Relay Shield es un módulo inteligente compatible con Arduino que cuenta con 4 relés mecánicos que proporcionan una forma fácil de controlar alto voltaje. La potencia máxima de conmutación es de 35VDC 70W para cada canal. Puede ser controlado directamente por Arduino/Seeeduino a través de E/S digital.

Aquí se lista la comparación entre varias versiones del Relay Shield:

|Parámetro| Relay Shield V1.0 |Relay Shield V2.0 |Relay Shield V3.0|
|---------|-------------------|-------------------|-----------------|
|**Voltaje de Trabajo** |+7V ~+12V |+5V |+5V|
|**Alimentación**| Fuente de alimentación externa |Pin 5V de Arduino/Seeeduino| Pin 5V de Arduino/Seeeduino|
|**Corriente de Conmutación (Máx)**| 2A |8A |8A|
|**Interfaz XBee** |Sí |No |No|
|**Control remoto**| Sí| No |No|
|**Shield Estándar**||No| Sí |Sí|
|**Indicadores de estado de trabajo para cada relé**| No| Sí| Sí|

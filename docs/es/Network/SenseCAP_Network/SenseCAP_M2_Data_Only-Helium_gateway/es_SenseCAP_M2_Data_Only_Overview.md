**Qué es SenseCAP M2 Data Only**

SenseCAP M2 Data Only es un Gateway LoRaWAN conectado a la Red Helium LongFi y capaz de proporcionar millas de cobertura de red inalámbrica y capacidad de transmisión de datos para dispositivos de Largo Alcance/LoRaWAN.

Como gateway LoRaWAN, podría ser capaz de proporcionar millas de cobertura de red inalámbrica y capacidad de transmisión de datos para dispositivos de Largo Alcance/LoRaWAN.

Como gateway que soporta la red Helium LongFi, podría ayudarte a construir la red LoRaWAN local y desarrollar varias aplicaciones LoRaWAN a bajo costo mientras disfrutas de los beneficios de ser parte de la Red del Pueblo.

SenseCAP M2 ahora está proporcionando versiones US915 y EU868 con certificados FCC y CE. Y tendrá más versiones de bandas de frecuencia y certificados correspondientes disponibles para diferentes países y regiones en el futuro cercano.

**Características**
============

![](https://www.sensecapmx.com/wp-content/uploads/2022/07/sensecap-m2-DO-features.png)

**· Cobertura de Amplio Rango y Señal Fuerte**

Con una Antena de Alto Rendimiento de 3dBi, SenseCAP M2 podría proporcionar millas de cobertura LoRaWAN y una señal fuerte. Y siempre podrías actualizar la antena para adaptarse a tu escenario de uso basado en diferentes entornos.

**· Rendimiento Excelente y Estable**

SenseCAP M2 está alimentado por la solución de hardware madura MT7628 y el chip de banda base de Largo Alcance Semtech SX1302/SX1303. Tiene BLE integrado y soporta conexión a internet tanto por Wi-Fi como por Ethernet. Para usuarios que necesitan alimentar el gateway por Ethernet en lugar de un cable de alimentación extra, la característica PoE también está añadida en M2.

**· Herramientas de Gestión Profesionales y Servicios en la Nube**

Los usuarios podrían configurar fácilmente el hotspot en unos pocos pasos simples a través de la App SenseCAP Hotspot. SenseCAP Hotspot Dashboard y SenseCAP Local Console también están desarrollados para que los usuarios monitoreen y gestionen el Hotspot de manera eficiente y fácil.

**· Soporte en Línea Eficiente y Oportuno**

Además de estas Herramientas de Gestión Profesionales y Servicios en la Nube, SenseCAP ha sido altamente elogiado por los clientes por su soporte técnico. Siempre que encuentres un problema técnico, el servicio de soporte técnico en línea eficiente siempre está aquí y te apoya de manera oportuna.

**· Seguro y Confiable**  
Para un gateway que maneja grandes cargas de transmisión de datos, la seguridad nunca es demasiada para preocuparse. SenseCAP M2 Data Only siempre asegura autenticación de alta seguridad y conectividad confiable a la Red del Pueblo.

**· Configuración Fácil en Pasos Simples**

Listo para usar con la App SenseCAP Hotspot en pasos simples

**· Accesorios Ricos**

Además de las características poderosas del dispositivo SenseCAP en sí, SenseCAP ha estado proporcionando opciones de accesorios ricos para que los usuarios obtengan el máximo rendimiento para los Hotspots. Tales como las mejores ventas de antenas IoT de alto rendimiento, cables de antena, almohadillas de montaje y arrestadores de rayos.

Este es el kit de breakout especialmente diseñado para ESP32, contiene una placa de breakout y conectores de pines. Las dimensiones son 53mmx38mmx1.2mm. La placa de breakout con conectores hembra de 2.54mm de separación te permite conectar los pines del ESP32 fácilmente. También proporcionamos 2 conectores grove en este kit para enriquecer la expansibilidad del ESP32.


[![enter image description here](https://files.seeedstudio.com/wiki/Seeed-WiKi/docs/images/get_one_now.png)](https://www.seeedstudio.com/ESP32-Breakout-Kit-p-2783.html)

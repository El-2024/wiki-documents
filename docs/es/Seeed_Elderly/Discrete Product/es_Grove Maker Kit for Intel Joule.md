Hace varias semanas publicamos el Grove Shield para Intel Joule. Esta vez, presentemos el increíble kit que puede ayudarte a comenzar con Intel Joule de manera fácil y rápida: el kit maker Grove para Intel Joule.

Joule es la placa más increíble que Intel publicó para proyectos IoT en 2016. Con el sistema Linux integrado y hardware rico, Intel Joule puede ser la mejor opción para que los desarrolladores mejoren su proyecto y publiquen el prototipo al mercado sin mucho esfuerzo.

El kit maker Grove para Intel Joule es el mejor kit para construir rápidamente prototipos basados en Joule. El kit contiene un shield bash grove para Joule, con el cual puedes agregar fácilmente los 13 módulos grove que seleccionamos específicamente para Joule. Si tienes el Intel Joule a mano y planeas construir un prototipo rápidamente, por favor no dudes en elegir el kit maker Grove para Intel Joule.

## Introducción de Partes

### [Grove Shield para Intel Joule](https://www.seeedstudio.com/Grove-Shield-for-Intel-Joule-p-2782.html)

![](https://files.seeedstudio.com/wiki/Grove_Maker_Kit_for_Intel_Joule/img/grove%20shield%20for%20intel%20joule.jpg)

El Grove Shield para Intel Joule es la clave para conectar el Intel Joule a la familia de módulos Grove. Simplemente conectándolo a tu placa Joule, puedes tener 8 conectores Grove sólidos y fáciles de usar que incluyen interfaces como I2C, UART, E/S digitales y entradas analógicas de inmediato. También mantiene los headers de pines 2x20 en caso de que quieras más GPIOs para el proyecto. Un interruptor está integrado en la placa para seleccionar el voltaje de trabajo a 5V o 3.3V.

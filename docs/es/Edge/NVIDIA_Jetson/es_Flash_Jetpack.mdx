### 4. Instalar dependencias de flash
Instalar las dependencias de flash en el PC Host:
```bash
sudo apt-get update && \
sudo apt-get install -y abootimg \
                        binfmt-support \
                        binutils \
                        cpp \
                        device-tree-compiler \
                        dosfstools \
                        lbzip2 \
                        libxml2-utils \
                        nfs-kernel-server \
                        python3-yaml \
                        qemu-user-static \
                        sshpass \
                        udev \
                        uuid-runtime \
                        whois \
                        openssl \
                        cpio \
                        rsync \
                        zstd
````

Adicionalmente, para Ubuntu 18.04 e inferiores, instalar `lz4c`

```bash
sudo apt-get install -y liblz4-tool
```

Para Ubuntu 20.04 / 22.04 y superiores:

```bash
sudo apt-get install -y lz4
```

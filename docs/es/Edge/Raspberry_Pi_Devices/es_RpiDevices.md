## HMI

### reTerminal

Conoce reTerminal, la última incorporación a nuestra familia reThings—un dispositivo avanzado de Interfaz Humano-Máquina (HMI) que cuenta con una pantalla multi-táctil capacitiva IPS de 5 pulgadas con una resolución de 1280 x 720. Este dispositivo preparado para el futuro se integra perfectamente con sistemas IoT y en la nube, permitiendo una colaboración eficiente para desbloquear escenarios infinitos en el borde. Con sus características de vanguardia, reTerminal está listo para redefinir las experiencias interactivas y cerrar la brecha entre los mundos físico y digital.

### reTerminal DM

Presentamos reTerminal DM, una Interfaz Humano-Máquina (HMI) industrial de código abierto de 10.1" que sirve como un Maestro de Dispositivo Integrado. Este dispositivo todo-en-uno, basado en Raspberry Pi CM4, funciona como un Panel PC, HMI, PLC y Gateway IIoT. Con una pantalla grande de grado industrial IP65, reTerminal DM es un hub de detección interactivo de próxima generación, optimizando el flujo de datos y la gestión de dispositivos en sitio.

## Controlador Edge

### reComputer R1000

El reComputer R1000 es un controlador IoT edge de alto rendimiento y costo efectivo basado en Raspberry Pi. Tiene interfaces Ethernet duales, tres interfaces RS485 aisladas, y soporta protocolos Modbus UDP/TCP y BACnet, haciéndolo ideal para edificios inteligentes y gestión de energía, incluyendo HVAC, iluminación, sensores y control de acceso. También ofrece amplias capacidades inalámbricas, incluyendo 4G, LoRa®, Zigbee, Wi-Fi y BLE.

## AI Box

### reComputer AI R2000

El reComputer AI R2000 es una poderosa computadora Edge AI construida sobre el Raspberry Pi 5, con un CPU quad-core Cortex-A76, 8GB RAM y soporte para SSD M.2. Con un acelerador AI Hailo-8 que entrega 26 TOPS, permite inferencia AI en tiempo real y baja latencia. Ofrece HDMI 4K, USB 3.0, Ethernet, Wi-Fi y Bluetooth 5.0 para conectividad perfecta. Ideal para análisis de video con AI, visión artificial y computación edge, trae AI de alta eficiencia al edge.

Esta carcasa está construida con aluminio de grado militar y acrílico resistente, ¡proporcionando la máxima protección contra daños físicos y corrosión! La caja esqueleto permite espacio libre para cualquier Arduino estándar, y las carcasas son apilables en las tres dimensiones, permitiendo posibilidades de expansión ilimitadas. El esqueleto es fácil de usar, ¡y proporciona la mejor protección para tu Arduino que el dinero puede comprar! Y esta es una caja simple, especialmente diseñada para BeagleBone®.
